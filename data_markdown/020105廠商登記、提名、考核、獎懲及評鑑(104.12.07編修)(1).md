## 內容摘要

本文件說明廠商登記、提名、考核、獎懲及評鑑之管理目的、適用範圍、公司規程、作業流程與評等標準。內容涵蓋新廠商登記、提名廠商選商、施工中定期考核、獎懲與年度評鑑等制度，適合作為採購發包、廠商管理與履約考核之 RAG 檢索資料。

## 1. 目的

為有效篩選專業、績優廠商及淘汰素質不良廠商，本公司訂定辦理採購案件（含作頭分包）之廠商登記、提名、考核及獎懲等作業要點，以評鑑方式達成合作廠商群之精實、可靠及信賴效益。

## 2. 定義及範圍

### 2.1 適用對象

本公司各工務所或開發所（以下簡稱工務所）辦理採購案件，提名廠商須選擇經過審核之合格廠商。

### 2.2 評鑑與管理機制

- 各工務所於每次計價辦理評核。
- 採購發包處於每年 10 月彙整當年度廠商考核成績。
- 經「廠商評鑑委員會」定期選拔審核優良廠商。
- 另可不定期且即時核定廠商停權處置名單。

## 3. 公司規程

1. 本公司「國內採購作業辦法」
2. 本公司「採購案件選商施行細則」
3. 本公司「廠商考核評鑑作業要點」

## 4. 流程及作業說明

## 4.1 新廠商登記流程

### 4.1.1 流程摘要

1. 填寫廠商基本資料表
2. 採發處審核
3. 登錄廠商資料庫
4. 完成

### 4.1.2 填寫廠商基本資料表

- 凡願參加本公司各項工程比價或議價之廠商，均應按照本公司「廠商登記須知」申請登記。
- 申請登記廠商應先詳實填寫下列表單：
  - 廠商登記表
  - 個人資料提供同意書
  - 切結書
- 表單應加蓋廠商及負責人印鑑，且須與最新版公司設立（變更）登記表上印鑑相同。
- 填妥後向總公司採購發包處或工務所申請登記。

### 4.1.3 採發處審核

廠商登記表資料由工務所或採購發包處核對收件後，進行綜合性審查評等，包含：

#### 書面審核
- 廠商基本資料
- 財務徵信
- 實績查核

#### 財務徵信補充
- 必要時得請財務處協助調查。

#### 實績查核
- 包括業主及同業對該廠商之評價。

#### 實地訪查
- 訪查廠商目前在手工程現場之：
  - 品質
  - 工安
  - 人員
  - 機具設備

### 4.1.4 登錄廠商資料庫

- 經審定合格之廠商，由採購發包處通告於公司內部網站，供採購發包人員使用。

## 4.2 選商提名、考核、獎懲及評鑑流程

### 4.2.1 流程摘要

1. 選商及提名
2. 決標廠商
3. 履約廠商
4. 定期考核
5. 評等
6. 依評等結果進行獎懲、停權或列入特優／優良廠商候選資格

### 4.2.2 選商及提名

#### 提名作業核定權責

| 採購案件等級 | 工務所 | 事業部主管 |
|---|---|---|
| 一級案件 | 核定 |  |
| 二級案件 |  | 核定 |
| 三級案件 |  | 核定 |
| 四級案件 |  | 核定 |

#### 提名原則

1. 工務所提名廠商後，報請主管處審核，再陳權責主管核定。
2. 權責主管得就經評鑑為乙等以上，或具備施工能力之適合廠商中增提參與競標。
3. 工務所若提名評鑑為丙等廠商時，須附工務所及主管處之保薦書。
4. 提名廠商前應徵詢其投標意願，以免受邀廠商臨時不參加投標。
5. 採比價或議價方式辦理時，選商對象原則應以本公司登記審查合格者優先。
6. 若因工程需要，可說明理由並經主管部同意後，提名具施工能力之合適廠商參加比價或議價。
7. 採購案件辦理應依「國內採購作業辦法」及「採購案件選商施行細則」辦理。

#### 提名廠商基本資料彙總表（範例重點）

- 廠商基本資料來源：
  - 營管系統
  - 經濟部商業司網站
- 可查詢項目：
  - 廠商登記等級
  - 重要相關工程實績（與中工有關為主）
- 表單簽認：
  - 提名人填寫
  - 所長、內業站長及承辦簽認

### 4.2.3 履約廠商

- 得標廠商收到決標函七日內，應至工務所辦理訂約及履約保證手續。

### 4.2.4 施工中定期考核

#### 考核時機

- 廠商考核由各工務所於每次計價完成後負責評核。

#### 第一次計價評核應登錄項目

工務所於廠商每個合約第一次計價評核時，須登錄：
1. 工班負責人及聯絡電話
2. 現場負責人及聯絡電話

若後續計價評核無異動則免填。

#### 特殊情事說明

- 若遇特殊情事，請簡述說明。
- 若無則免填。

#### 考核標準

協力及供應廠商評鑑考核標準分為三大項：
1. 進度
2. 品質
3. 工安

其中進度面向包含施工進度、配合度、專業度、技術層面、施工管理、溝通協調、財務能力、規劃能力、履約能力、商譽、誠信等。

#### 低分處理原則

- 若分數低於乙等，工務所及各主管部門應立即提出改善計劃並督促改善。
- 對累計 2 次考核分數低於乙等之廠商，應立即研議終止合約或協助繼續施工等處理方式，並簽報採購發包處，轉請總經理召開「廠商評鑑委員會」討論後實施。

### 4.2.5 評等

#### 考核計算方式

```text
廠商各合約、各工班考核 =
(第一次進度考核 × 50% + 第一次品質考核 × 20% + 第一次工安考核 × 30% + 第二次進度考核 × 50% + ...)
÷ 累計考核次數
```

```text
廠商考核成績 =
(合約一平均考核 + 合約二平均考核 + 合約三平均考核 + ...)
÷ 累計合約數
```

#### 評等等級

| 等級 | 標準 |
|---|---|
| 特優 | 連續二年考核成績均在 95 分（含）以上 |
| 優等 | 平均 90 分（含）以上 |
| 甲等 | 平均 80 分（含）以上，未滿 90 分 |
| 乙等 | 平均 70 分（含）以上，未滿 80 分 |
| 丙等 | 平均 70 分以下 |

### 4.2.6 獎勵與懲處

#### 優等廠商獎勵
1. 可取得優良協力廠商及供應廠商資格
2. 寄發優良廠商通知函
3. 主辦單位得優先提名參加比價或議價
4. 降低投標押標金及履約保證金 50%

#### 特優廠商獎勵
1. 可取得特優協力廠商及供應廠商資格
2. 寄發特優廠商通知函
3. 提名參加議價
4. 降低投標押標金及履約保證金 75%

#### 其他處理
- 配合金質獎、金安獎得獎廠商，得比照優等廠商辦理。
- 甲、乙等者，不獎勵也不處罰。

#### 列入拒絕往來廠商之情形
1. 廠商經評鑑為丙等者
2. 承攬本公司分包工程發生重大職業災害及品質嚴重瑕疵事件，經主管處認定奉准簽核者
3. 高風險或停權期限內廠商，須加提施工安全衛生計畫經核定後始准提名

### 4.2.7 年度評鑑與公告

1. 採購發包處於每年 10 月彙整當年度廠商考核成績。
2. 經「廠商評鑑委員會」選拔優良廠商，並不定期且即時核定停權處置名單。
3. 優良廠商考核評鑑每年舉辦一次，有效期為二年。
4. 年度評鑑結果，經總經理核定後公布。

## 5. 文件及格式

1. 廠商提名、考核及評鑑作業檢核表
2. 提名廠商基本資料彙總表
3. 廠商登記表、個人資料提供同意書及切結書
