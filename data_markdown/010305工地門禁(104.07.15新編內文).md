# 010305 工地門禁

## 文件中繼資料

- 文件名稱：工地門禁
- 作業流程節次：010305
- 版本日期：104.07.15 新編
- 來源類型：中華工程股份有限公司工務所管理作業手冊
- RAG 目標：保留門禁目的、定義、管制流程、人員與車輛機具進場規定、隧道工程門禁範例與其他安全管制，並提供圖像索引與結構化檢索欄位。

## 1. 目的

### 1.1
為工務所有效管理工地安全門禁管制，確保工地安全，訂定之。

### 1.2
依據承辦工程之相關合約規範、法令規定及本公司規章辦理。

## 2. 定義及範圍

### 2.1 範圍

適用本公司承辦工程所轄之：
- 工作者（同仁）
- 各協力廠商（含再承攬人）
- 訪客
- 供應商

### 2.2 定義

#### 2.2.1 管制內容

1. 工地作業人員行為均應接受規範，遵守工地工作守則。
2. 工地作業人員之基本資料應列冊建檔。
3. 進入工地之作業人員應配帶工作識別證，以利識別。
4. 工地內之人員及車輛攜運物品應隨時接受檢查。
5. 工地內嚴禁攜帶武器及違禁物品，並禁止飲用酒精性飲料、賭博及破壞施工材料設備等行為。
6. 遇有突發語言、肢體、精神等違法情事，應即向當地治安機關及有關單位通報。
7. 工地之安全設施應隨時加以維護。

## 4. 流程及作業說明

### 4.1 人員進場管制流程

#### 4.1.1 流程摘要

人員進場申請 → 書面審核 → 危害告知 → 簽署紀律承諾書 → 遵守工作守則 → 核發識別證或帽貼 → 出工進場作業 → 領班／工程師抽查

#### 4.1.2 書面審核應備資料

| 項目 | 說明 |
|---|---|
| 勞健保 | 需備妥勞保、健保資料 |
| 體檢表 | 需有體格檢查資料 |
| 教育訓練合格證明 | 需具備相關安全教育訓練證明 |

#### 4.1.3 缺件處理原則

| 狀況 | 處理方式 |
|---|---|
| 缺 1 | 補件後再審 |
| 缺 2、3 | 補件後再審 |
| 無教育訓練、體格檢查 | 先實施一般勞工教育訓練、體格檢查 |

#### 4.1.4 人員進場申請與識別管理

人員進場前應完成：
- 書面審核
- 危害告知
- 簽署紀律承諾書
- 核發識別證或帽貼

#### 4.1.5 勤務管制與職責

1. 工地周圍應設置固定圍籬及警告標示，門禁管制警衛由保全公司或工地自雇人員擔任。
2. 工區由單一大門進出，後門或邊門平日上鎖管制。
3. 門禁管制人員應定時巡邏，至少每 2 小時一次，必要時可用行車紀錄器紀錄巡視過程。
4. 門禁管制人員應主動檢查出場車輛車斗、行李箱並核對放行單註記之品名、數量。
5. 重要材料、器材、機械、設備集中堆放處應加強戒護。
6. 嚴禁夜間及假日帶料出場、領料或搬料，緊急狀況須經工務所簽單確認始可放行。
7. 設定夜間巡邏點簽到，並以對講機回報工務所值班人員。

#### 4.1.6 門禁管制作業與留存記錄

| 記錄類型 | 說明 |
|---|---|
| 工作者進場作業識別證發放記錄 | 識別證發放與核對 |
| 人員進出場管制簽到／簽退／刷卡記錄 | 進出時間控管 |
| 車輛／施工材料進出場記錄 | 車輛與物料流向控管 |
| 危險性機械進出場記錄 | 一機三證與設備進出場控管 |

#### 4.1.7 工程施工地區人員管制規定

1. 凡進出施工地區一律須配帶工務所製發之識別證，否則禁止進入。
2. 業主單位人員因業務需要，可憑單位製發之識別證出入工地。
3. 臨時性一般洽公或會客，憑身分證或駕照換取來賓證進入，離場時換回原件。
4. 進場施工人員一律需經查核參加勞、團保，始准進入施工場所。
5. 不得使用聘僱未經許可、許可失效或其他單位所申請聘僱之外籍勞工。
6. 協力廠商應提供進場施作所須之相關資料供工務所查核。
7. 經查核人員中有重大素行不良者，協力廠商應無條件將其調離工地。
8. 識別證遺失應登記作廢及補發，並罰款新台幣參佰元整。
9. 識別證轉借他人使用，經發現，除沒收外，並罰款壹萬元及依法追究。
10. 特定區內除輪值人員及庫房人員外，禁止留宿及開伙。
11. 進入工區一律要戴安全帽、反光背心及穿著安全鞋；高處作業者應配戴適當安全帶。

### 4.2 車輛機具進場管制流程

#### 4.2.1 流程摘要

車輛機具申請 → 書面審核 → 核發通行證 → 機具進場 → 警衛檢查證照 → 機具現場作業 → 現場工程師／安衛人員抽查

#### 4.2.2 一般車輛管制規定

1. 工務所及協力廠商之車輛進出工區時，應憑工務所製發並向業主監造或相關單位報備之車輛通行證。
2. 車輛裝載物品須符合規定，並配合門禁管制人員檢查。
3. 臨時性車輛進出工區時，應由工務所人員引導進入工地。
4. 車輛通行證不得轉借，違者沒收並依規定究辦。

#### 4.2.3 危險性機械進場申請

危險性機械進場應查核「一機三證」：

| 類別 | 內容 |
|---|---|
| 車 | 合格證、申請檢驗 |
| 人 | 操作手參訓／合格證 |
| 指揮手 | 吊掛指揮手參訓／合格證 |

#### 4.2.4 危險性機械現場查核重點

1. 警衛檢查證照。
2. 現場工程師／安衛人員抽查。
3. 檢查三證有效期限。
4. 確認自主檢查表、檔樁、枕木等作業條件。

### 5.2 門禁管制作業原則

工地現場應設置警衛亭及警衛，管制施工人員及機械（車輛）之進出。人員（車輛）進入工區時，警衛應主動詢問辦理內容，通報工務所人員確認是否可進入，填寫登記完成後始准進入。

危險性機械進場時，警衛應確實查核一機三證：
- 機械檢驗合格證
- 操作手合格證
- 吊掛指標手合格證

不合格者禁止進入作業。

### 5.3 隧道工程門禁管制（範例）

| 步驟 | 說明 |
|---|---|
| 步驟 1 | 人員進出隧道內簽到管制作業 |
| 步驟 2 | 出入坑管制感應系統作業 |
| 步驟 3 | 感應後自動核對顯示 in/out 狀態 |
| 步驟 4 | 入坑管制翻牌作業 |

### 5.4 其它安全管制

1. 禁止外賓留宿。
2. 禁止賭博、酗酒。
3. 施工作業區內一律禁止會客（工務所辦公區除外）。
4. 倉庫物料存儲處所，非經許可禁止進入。
