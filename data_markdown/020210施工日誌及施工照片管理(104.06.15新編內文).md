# 020210 施工日誌及施工照片管理

## 1. 目的

藉由現場工程師每日之施工日誌及施工照片詳實記錄工地現場實際施工情況，自我檢討負責工項是否處於落後、使用物料是否超出設計、人力機具安排是否過於浮濫、施工現場是否存在不可抗拒等潛在因素影響工進，並於每日收工後妥善安排隔日進場施工項目、人力機具或解決方案，並讓主管了解工地狀況，必要時可協助解決，以確保每日工進等相關事宜。

工務所內業相關負責人員另於隔日匯集各主站長及現場工程師日報後，整合施工內容、工項、數量、人員、機具、材料及其他等重要事項，作為施工管理參考依據、業主及小包計價佐證資料、提高工程管理效益，同時也為日後爭議或施工改進留下寶貴記錄。

## 2. 定義及範圍

### 2.1 定義

| 編號 | 名稱 | 說明 |
|---|---|---|
| 2.1.1 | 施工日誌 | 即施工現場之工作日記，每日記載工地現場施工狀況及異常或重大事件 |
| 2.1.2 | 施工照片 | 利用拍攝照片，藉以記錄各相關作業施工過程前、中、後之情形及比較 |

### 2.2 範圍

從工程開工至竣工期間，均應完整每日確實填寫及拍照紀錄。

## 3. 流程及作業說明

### 3.1 施工日誌流程及作業說明

#### 3.1.1 流程總覽

主要流程如下：

1. 現場工程師填寫工程日報表
2. 施工站長檢核項目及數量
3. 數量與記事確認
4. 所長或副所長審核
5. 內業站輸入 PO-ERP 系統
6. PO-ERP 系統轉出工程施工日誌

#### 3.1.2 現場工程師填寫工程日報表

1. 各工程師每日下班前填寫完成後送交施工站長檢核。
2. 工程日報表應詳細填載：
   - 重要施工項目
   - 數量
   - 出工人數
   - 使用機具

#### 3.1.3 施工站長與所長審核

1. 施工站長檢核項目、數量及記事無誤後，送所長或副所長審核。
2. 所長或副所長應確認：
   - 數量
   - 重要記事
   - 影響施工之重要事項記錄

#### 3.1.4 內業站輸入 PO-ERP 系統

所長或副所長審核無誤後，由內業站輸入 PO-ERP 系統產出報表。

#### 3.1.5 未導入 PO-ERP 之替代作法

尚未導入 PO-ERP 系統或無需經營管系統產出報表之工地，可直接輸入公共工程施工日誌 Excel 表提送業主或監造。

#### 3.1.6 PO-ERP 轉出施工日誌內容

PO-ERP 系統轉出工程施工日誌時，應完整包含：

| 欄位 | 內容 |
|---|---|
| 天氣及進度 | 當日施工天候與進度 |
| 施工項目及數量 | 當日完成工項及數量 |
| 出工人員、機具及使用材 | 人員配置、使用機具與材料 |
| 每日施工重要事項記錄 | 當日重大事件、異常、指示與處理情形 |

#### 3.1.7 填寫施工日誌注意事項

施工日誌格式可參考公共工程委員會之制式版本，原則上應包含制式版本所列欄位，亦需配合各業主機關依工程性質及契約約定事項自行增訂之格式。

施工日誌原則應包含：

- 重要施工項目完成數量
- 供給材料使用數量
- 出工人數
- 使用機具
- 工程進度
- 施工取樣試驗
- 通知承商辦理事項
- 重要事項紀錄

#### 3.1.8 重要事項紀錄內容

重要事項紀錄應包含：

| 類別 | 內容 |
|---|---|
| 主辦機關及監造單位指示 | 當日交辦事項 |
| 緊急異常狀況 | 工地異常之通報與處理情形 |
| 第三人會勘或會議 | 協調與會勘記錄 |
| 專任工程人員督察 | 是否按圖施工、解決施工技術問題等 |

#### 3.1.9 專業工程特定施工項目之日誌要求

專業工程特定施工項目之施工日誌應另記載：

1. 當日施工之各項專業工程特定施工項目內容。
2. 各特定施工項目僱用技術士姓名。
3. 技術士證書字號。
4. 該技術士之簽名或蓋章。

#### 3.1.10 停工期間填寫原則

於工程停工期間，施工日誌仍應填寫當日工作事項，例如：

- 施工圖繪製
- 計畫書撰寫
- 材料準備
- 機具保養

停工期間工作日誌應註明：

| 項目 | 要求 |
|---|---|
| 本日完成進度 | 0% |
| 要徑項目 | 勿填寫 |
| 停工原因 | 應詳實列入重要記事欄 |
| 特殊情事 | 應詳實記錄 |

#### 3.1.11 其他事項

重大事項如需與工務所主管溝通者，亦應明確記錄於施工日誌。

### 3.2 施工照片流程及作業說明

#### 3.2.1 流程總覽

主要流程如下：

1. 工程師現場拍照
2. 工項、勞安檢核
3. 照片資料分類存檔
4. 工務所建立共用照片資料夾
5. 每週上傳營管系統照片（至少 12 張）
6. 分類存檔

#### 3.2.2 現場拍照原則

現場工程師應就下列內容拍照存檔：

- 工地地形
- 地貌
- 各施工工項之檢查點

#### 3.2.3 照片內容要求

施工照片應明確表現：

| 要素 | 說明 |
|---|---|
| 人 | 參與作業之人員 |
| 事 | 施工或查驗事項 |
| 時 | 拍攝日期時間 |
| 地 | 施工位置 |
| 物 | 構件、材料、設備、工項內容 |

並應注意符合勞安法規要求。

#### 3.2.4 共用資料夾與分類存檔

1. 工務所應於電腦主機共用資料夾中設立專區存放施工照片。
2. 照片應分類註記存檔備查。
3. 應建立施工照片共用資料夾，供各工項分類管理。

#### 3.2.5 每週上傳要求

1. 指派專人每週彙整重要工作項目並建檔。
2. 所有照片均應顯示日期。
3. 應詳細記錄照片資料內容。
4. 應將現場主要施工照片上傳營管系統至少 12 張。
5. 已建置 PO-ERP 之工地，應上傳至 PO-ERP 系統。

#### 3.2.6 一般工項與特殊事項記錄原則

1. 一般工項依固定原則拍攝與記錄。
2. 特殊事項除記載於施工日誌外，亦應列入施工照片記錄。
3. 照片說明宜以文字加註，便於後續查找與舉證。

#### 3.2.7 特殊狀況拍照存證

施工中遇有下列特殊狀況時，應於施工前、中、後拍照存證並專案存檔：

| 類型 | 說明 |
|---|---|
| 湧水 | 地下水或突發湧水狀況 |
| 特殊地質 | 地質異常或與原設計不同情況 |
| 地下管線 | 既有地下設施影響 |
| 隱蔽部份 | 施工完成後不易再查驗之部位 |
| 地下有價埋藏物 | 需額外處理之埋藏物 |
| 危險物品 | 施工現場發現之危險物 |
| 工程災害 | 施工中災害或事故 |
| 鄰近建築結構危害 | 附近建築或構造物受影響 |
| 抗爭事件 | 社會抗爭或租地爭議等事件 |

#### 3.2.8 配合業主、監造查驗之照片要求

配合業主、監造之查驗照片時，其項目、地點、日期必須標註清楚，以利工程計價請款佐證；如業主、監造有其他要求，則依其意見修正辦理。

業主查驗照片宜包含：

- 標註施工項目
- 標註地點
- 標註日期
- 相關查驗人員配合入鏡
