## 1. 目的

### 1.1 安衛管理目的
為使同仁熟悉施工中安衛管理，有效掌握承攬廠商管理，避免施工中觸及安衛法令，並透過安全衛生組織、人力資源與制度計畫的有效運用，降低爭議發生時對公司造成之財務損失。

### 1.2 施工前與施工中應編訂或修編之文件
施工前編訂、施工中修編下列文件，作為全體施工人員遵循依據：

- 職業安全衛生管理計畫書
- 交通維持計畫書
- 逕流廢水污染削減計畫書
- 施工環境保護執行計畫書
- 丁類場所危險性工作場所評估計畫書
- 工務所每月 / 每季傳總公司安衛資料

### 1.3 TOSHMS 系統要求
工務所應依 TOSHMS 職安衛系統要求事項，建立、文件化、實施、維持並持續改善安全衛生管理系統，並界定其適用範圍。

## 2. 定義及範圍

### 2.1 範圍
本作業適用於本公司承辦工程之：

- 工安相關規劃
- 工地安全衛生內外業文件管理
- 門禁管理作業
- 敦親睦鄰相關事項

### 2.2 定義

#### 2.2.1 職安衛系統 TOSHMS
對工程危害鑑別、風險評估、控制措施、文件管制、法令規章、目標與方案、能力、訓練、認知、作業管制、緊急應變、檢查、績效量測、事件調查、不符合事項、矯正與預防措施、紀錄管制、內部稽核、管理階層審查等所訂定之安全衛生管理實務、資源與作業程序文件。

#### 2.2.2 職業安全衛生管理計畫
針對特定工程性質、施工設備與環境因素所訂定之安全衛生管理文件，可依工程需要選擇章節製訂，例如：

- 工程概要
- 安全衛生組織及人員
- 安全衛生工作守則
- 職業安全衛生教育訓練計畫
- 緊急或意外事故應變措施
- 自動檢查計畫
- 環境保護管理計畫
- 電器設備安全計畫
- 防颱與防汛計畫
- 開挖作業計畫
- 交通管制計畫
- 火災防護計畫
- 作業環境檢測計畫
- 工作梯、施工架及工作台管制計畫
- 出入許可計畫

#### 2.2.3 交通維持計畫
針對特定工程性質、施工設備與環境因素所訂定之交通維持管理文件，主要章節包括：

- 工程概要
- 周邊道路現況
- 施工中交通維護方案

#### 2.2.4 逕流廢水污染削減計畫
依規定以下列章節製訂：

- 工程概述
- 環境背景說明
- 預防及管理措施
  - 污染控制目標
  - 污染控制方法及污染控制設施
  - 人員訓練計畫
  - 污染控制監測
  - 污染控制效果概估
  - 維護管理計畫

#### 2.2.5 施工環境保護執行計畫
以下列章節製訂：

- 開發內容與工程概述
- 環境保護措施
- 施工階段環境監測計畫
- 環境管理組織
- 環境管理監督及查驗作業計畫
- 施工期間環境保護經費

#### 2.2.6 環境監測計畫
依施工計畫書及環評承諾事項辦理監測工作，並針對異常情形提出檢討與施工因應對策。主要章節包括：

- 計畫內容
- 計畫執行之組織及職責
- 檢驗數據之品保目標
- 採樣作業方法
- 樣品管制
- 校正程序及頻率
- 分析程序
- 數據之演算、驗證及報告
- 內部品質管制檢查及頻率
- 內部稽核
- 預防性維護
- 評估數據品質目標
- 改正措施之依據及步驟
- 工作報告格式及內容綱要

#### 2.2.7 丁類場所危險性工作場所評估計畫書（危評計畫）
危評計畫適用於下列營造工程：

| 編號 | 適用條件 |
|---|---|
| 2.2.7.1 | 建築物高度在五十公尺以上之建築工程 |
| 2.2.7.2 | 橋墩中心與橋墩中心之距離在五十公尺以上之橋樑工程 |
| 2.2.7.3 | 採用壓氣施工作業之工程 |
| 2.2.7.4 | 長度一千公尺以上或需開挖十五公尺以上豎坑之隧道工程 |
| 2.2.7.5 | 開挖深度達十五公尺以上或地下室為四層樓以上，且開挖面積達五百平方公尺之工程 |
| 2.2.7.6 | 模板支撐高度七公尺以上，面積達一百平方公尺以上且佔該層模板支撐面積百分之六十以上者 |

危評計畫主要章節：

- 工程概要
- 施工安全評估報告書
- 施工計畫

> 危評計畫非經勞動檢查機構審查合格或檢查合格之工作場所，不得使勞工在該場所作業。

## 4. 流程及作業說明

### 4.1 安衛管理總流程

安衛管理流程涉及以下角色與機制：

- 總公司職業安全衛生委員會
- TOSHMS 管理審查會
- 職業安全衛生管理處
- 工務所每日安衛管理循環
- 安全衛生協議組織
- 內部稽核
- 共同維護作業安全、邁向零災害

### 4.2 總公司職業安全衛生委員會

- 每三個月由主任委員召開一次，必要時得召開臨時會議。
- 委員七人以上，任期兩年。
- 委員組成包括：
  - 職業安全衛生管理人員
  - 相關部門主管
  - 從事勞工健康服務之醫護人員
  - 安全衛生有關工程技術人員
  - 本公司工會選舉代表，且應佔委員人數三分之一以上
- 決議報請公司核交有關單位執行。
- 會議紀錄除公布外，應留存供勞動檢查機構查核。
- 第四季職安委員會會同 TOSHMS 管理審查會，彙報執行成果。

#### 安衛委員會文件範例
- 安衛委員提案表
- 開會通知
- 會議記錄
- 會議記錄發佈

#### 協議組織主要功能

- 職安衛作業指導及協助
- 職災防止宣導、訓練與教育
- 作業場所巡視缺失檢討改進

#### 會議頻率
- 原則上每月召開一次
- 必要時得召開臨時會議

### 4.11 內部稽核（每月）

- 職安處每月赴各工務所實施現場安全衛生檢查兩次。
- 對高風險或重大職災工地嚴格督導持續改善。
- 若缺失過多、改善遲緩、發生職災或重大事故，所長應至職業安全衛生委員會進行改善與事故檢討報告。
- 各單位應將總公司檢查紀錄納入「安全衛生管理成效表」統計。

## 6. 其他安全衛生管理事項

### 6.1 TOSHMS / OHSAS18001 職安衛系統工務所、承攬商應辦事項

| 項次 | 應辦事項 | 頻率 / 時機 / 說明 |
|---|---|---|
| 1 | BES-4-T01-03-職安衛風險評估表（含辦公室內外、倉庫等環境） | 每月評估三個月內施工工項之風險評估等級並傳送職安處 |
| 2 | BES-4-T01-04-職安衛目標管控表 | 每年 12 月底前訂定工務 / 開發所新年度安衛目標並傳送職安處 |
| 3 | BES-4-T01-05-管理方案規劃表 | 每季規劃並傳送職安處 |
| 4 | BES-4-T01-17-採購安衛需求表 | 辦理工程發包或採購機械、器具、設備、原料及個人防護具等需求時 |
| 5 | BES-4-T01-02-法令符合性評估表 | 職安處追蹤更新發佈，由工務 / 開發所依法令適用性評估工地現況 |
| 6 | BES-4-G02-01-安衛資格管理表 | 每半年 6 月下旬、12 月下旬送交職安處備查，作為證照及回訓管制依據 |
| 7 | BES-4-T01-27-安衛績效統計表 | 每年元月訂定並傳送職安處 |
| 8 | BES-4-T01-18-安衛設備清冊 | 安衛設備採購、驗收時紀錄 |
| 9 | BES-4-T01-19-領用登記表 | 同仁 / 承攬商領用時紀錄 |
| 10 | BES-4-T01-21-變更風險評估表 | 發生新工程 / 新材料 / 新設備 / 新工法 / 意外事件 / 法令變動 / 風險管制不足 / 管理代表或總經理指示時辦理 |
| 11 | BES-4-T01-16 安衛缺失通知單 | 不合格時開立通知單（含照片）交承攬商改善，並追蹤至完成 |
| 12 | BES-2-G01-安衛文件記錄管理流程 | 管理文件系統架構、制修訂、編碼、識別、保存期限等 |
| 13 | BES-2-A02 安衛內部稽核流程 | 配合職安處辦理內稽 |
| 14 | BES-3-作業標準彙總（等 13 項） | 各承攬廠商施工作業前應提供廠商參考執行 |

### 6.2 工務 / 開發所傳總公司職安處安衛資料

| 編號 | 項目 |
|---|---|
| 6.2.1 | 每月 5 日前傳送各專案「安全衛生管理成效表」、「責任工區畫分表」、「TOSHMS 職安衛風險評估表」 |
| 6.2.2 | 每月 5 日前至營建資訊管理系統填報「工務所工安環保月報表」 |
| 6.2.3 | 勞檢單位巡查後，填報聯合稽查工地安全衛生檢查紀錄表清單，並回傳缺失改善電子檔 |
| 6.2.4 | 每週五填報「工務所現場監督人員巡視檢查紀錄管控表」 |
| 6.2.5 | 依 TOSHMS 應繳資料說明，依年 / 季 / 月頻率報備系統文件 |
| 6.2.6 | 新開工或已動工工務所應將安衛計畫書、危評計畫、工作守則、安衛人員設置報備書等報備職安處 |
| 6.2.7 | 每半年傳送工務 / 開發所工安績效獎金名單 |
| 6.2.8 | 每年底傳送工務所安衛師 / 員、營造業安衛業務主管設置報備，核發工安證照租金 |
| 6.2.9 | 如遭勞檢單位罰款，工務所應提出責任檢討與改善檢討報告 |
