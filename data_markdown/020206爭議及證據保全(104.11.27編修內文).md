## 1. 目的

為適當解決契約執行爭議，維護公司權益，適時保全證據，提高本公司於爭議仲裁或訴訟之勝率，乃訂定本作業指引。

## 2. 定義及範圍

契約執行過程中，業主或協力廠商因契約條款解釋、指示、程序及變更，對其適法性產生歧見及爭議；其解決歧見及爭議之一切保全、協商、請求及法律上之調解、仲裁、訴訟等，均屬本作業範圍。

## 3. 流程及作業說明

### 3.1 業主爭議及證據保全作業流程

#### 3.1.1 流程總覽

主要流程如下：

1. 契約執行
2. 履約內容與契約規定不符
3. 書面資料通知業主
4. 變更設計
5. 爭議事件重要性判斷
6. 重要者提報主管處，依公司爭議案件作業流程準則辦理
7. 不重要者保全相關資料至完工再一併處理

#### 3.1.2 契約執行階段基本原則

1. 應了解工程契約條款，辦理契約管理之意思表示與事實蒐集。
2. 履約過程中如因合約條款、圖說或外在環境影響履約，致公司權益受損時，工務所應適時提出釋疑或請求。
3. 若業主駁回或不處理，工務所除再申覆外，應以不攻擊業主或顧問缺點、弱點之原則下，表達本公司持異議或不接受之立場。
4. 當請求給付時，需將理由說明清楚；若業主不同意，亦應要求業主明確表達不同意理由，以利成立爭議基礎。

#### 3.1.3 履約內容與契約規定不符

當履約內容與契約規定不符時，應先行確認：

| 項目 | 說明 |
|---|---|
| 不符態樣 | 例如契約規定為 A 型式，但履約階段業主要求以 B 型式施作 |
| 指示形式 | 屬業主正式書面指示或僅為口頭建議 |
| 時效要求 | 原則上三日內以書面請求確認是否為業主指示變更 |

#### 3.1.4 書面資料通知業主

1. 應依契約規定彙整書面資料提送業主。
2. 請求辦理契約變更事項，例如契約變更等。

#### 3.1.5 契約變更程序

1. 應依契約規定及相關規定辦理契約變更程序。
2. 若履約期間涉有爭議之案件，就爭議事項之相關會議或協商，如屬契約變更，工務所應邀請法務處派員參與。
3. 目的在於表達本公司重視該爭議案件，並主張契約權利，以避免工務所直接與業主或顧問正面衝突。

#### 3.1.6 爭議事件重要性判斷

應先行判斷：

| 判斷項目 | 說明 |
|---|---|
| 是否造成公司重大損害 | 若是，應立即列管處理 |
| 是否有時效風險 | 承攬報酬請求權時效為兩年，損害賠償為一年 |

若有上述情形之一者，應立即提出爭議事項。

#### 3.1.7 專案管理與提報主管處

若屬重要爭議案件，應：

1. 將爭議案件資料專檔管理。
2. 指定管理人彙整與爭議執行有關之通知、提示、會商、審查、施工、計價等文件及佐證資料。
3. 爭議索賠案件應在契約或法令規定期限內適時提出。
4. 提出前，工務所應與公司法務處研討商議對策及文件處理方式。
5. 應隨時更新資訊並掌握情事發展。
6. 若業主駁回或不處理，得再申覆後列入爭議案件專管追蹤。

#### 3.1.8 完工後一併處理之資料保全

若屬不重要案件，得保全相關資料至完工驗收後再一併處理，並將相關卷宗資料移交法務處辦理並協助說明。

##### 應保全之證據資料

| 編號 | 證據類型 |
|---|---|
| 1 | 估驗資料 |
| 2 | 檢（試）驗報告 |
| 3 | 監工日報、工地日誌或工作記錄 |
| 4 | 施工備忘錄及會議紀錄 |
| 5 | 往返函件 |
| 6 | 施工照片或影片及相關所有記錄 |
| 7 | 施工計畫及書圖送審記錄 |
| 8 | 工料分析成本及市場行情資訊 |

### 3.2 協力廠商爭議及證據保全作業流程

#### 3.2.1 流程總覽

主要流程如下：

1. 契約執行
2. 協力廠商提出契約外請求
3. 判斷請求事項是否符合契約規定
4. 工務所與協力廠商協商處理
5. 不符合則駁回請求並保全資料結案
6. 協力廠商提請爭議處理時，將爭議資料及建議方案簽報主管處，提請法務處協辦

#### 3.2.2 契約執行與請求事項判斷

1. 工務所應了解工程契約條款，辦理契約管理之意思表示與事實蒐集。
2. 先行確認履約內容與契約規定不符之處態樣為何。
3. 應判斷相關事項是書面指示或口頭指示。
4. 如確屬契約規定範圍內事項，應依契約規定辦理後續事宜。

#### 3.2.3 協力廠商請求不符契約時

若請求事項不符合契約規定，應：

1. 駁回請求。
2. 保全資料後結案。
3. 辦理專案專卷管理。

#### 3.2.4 專卷管理資料

協力廠商爭議案件之證據保全資料包括：

| 編號 | 證據類型 |
|---|---|
| 1 | 估驗資料 |
| 2 | 檢（試）驗報告 |
| 3 | 監工日報、工地日誌或工作記錄 |
| 4 | 施工備忘錄及會議紀錄 |
| 5 | 往返函件 |
| 6 | 施工照片或影片及相關所有記錄 |
| 7 | 施工計畫及書圖送審記錄 |
| 8 | 工料分析成本及市場行情資訊 |

#### 3.2.5 協力廠商提請爭議處理時

應：

1. 將爭議案件資料專案管理。
2. 彙整相關來往文件。
3. 檢整歷次會議記錄及相關書面資料。
4. 隨時更新相關資訊並掌握情事發展。
5. 將相關卷宗資料移交法務處辦理並協助說明。

### 3.3 工程索賠作業流程

#### 3.3.1 索賠構成要件

```text
索賠（claim）＝額外工作或工期＝實際之工作或工期－契約規定之工作或工期
```

#### 3.3.2 索賠處理主軸

工程索賠流程應依下列脈絡辦理：

1. 發生履約權益減損事件
2. 契約條款研判與法律實務分析
3. 事實原因調查與資料憑證收集
4. 索賠專案適時提出

#### 3.3.3 索賠資料整理方向

| 類別 | 內容 |
|---|---|
| 排除不利 | 排除不利於本公司之事實與證據缺口 |
| 尋找有利 | 找出有利於本公司之文件、照片、紀錄及指示依據 |
| 釐清爭點 | 明確界定爭議原因、責任歸屬與請求內容 |
| 確立訴求 | 明確提出價金調整、給付報酬、損害賠償、工期延長等訴求 |

#### 3.3.4 常用索賠證據來源

- 施工資料及會議紀錄
- 請求及申覆文件
- 往返文件
- 業主指示及圖說文件
- 事件佐證照片

#### 3.3.5 後續法律途徑

索賠後續處理可包括：

- 協商
- 調解
- 仲裁
- 訴訟

## 5. 其他事項

### 5.1 對分包商證據保全方面

1. 本公司為上市公司設有法務處，而分包商可能連法務人員都沒有，原則上本公司與分包商履約爭議之證據保全或留存紀錄較完整。
2. 在契約上或舉證上，本公司通常較有優勢，勝訴可能性亦較高。
3. 但工務所主管不應協助分包商舉證，以免反而造成公司敗訴之不利益。
4. 因此目前公司已將分包合約竣工驗收權限，排除在所長授權書內。
5. 顧問建議仍應載明驗收單位及授權之驗收人。

### 5.2 對業主證據保全方面

1. 比較欠缺之資料通常為施工照片及實支法之管理費用。
2. 於爭議或訴訟時，如有施工照片可佐證施工項目，較易說服委員或審判長。
3. 若工地展延工期請求管理費，至少從展延開始至結束，要蒐集實支費用收據，以資證明。
