## 1. 目的

工地經常業務或特定事項之需，由公司撥付工地週轉金零星使用，為使工務所於零用金運用及報銷均能符合公司內部作業及法令規範，乃訂定本作業之處理程序。

## 2. 定義及範圍

### 2.1 定義

| 編號 | 名稱 | 說明 |
|---|---|---|
| 2.1.1 | 固定零用金（定額週轉金） | 公司視工務所經常業務之需要，按核定額度撥付之款項。 |
| 2.1.2 | 臨時零用金 | 依特定事項所需支付款項超過固定零用金時，依其所需金額專案申請撥付之款項。 |

### 2.2 範圍

## 4. 流程及作業說明

### 4.1 流程總覽

1. 請購單
2. 核准
3. 採買
4. 支出憑證審核
5. 報銷資料製作
6. 列帳核銷

#### 4.1.2 請款回補流程

1. 回補請款
2. 支出傳票
3. 撥款
4. 工地銀行戶頭
5. 工地保管人提款
6. 回補款
7. 零用金循環使用

### 4.2 報銷流程與請款回補流程明細

#### 4.2.1 （a）請購、核准、採買、支出憑證審核、報銷製作

##### 4.2.1.1 （a-1）請購

使用營建管理資訊系統填寫請購單，重點包括：

| 欄位/動作 | 說明 |
|---|---|
| 填寫請購項目 | 明確描述採買品項。 |
| 填寫用途 | 說明支出用途與業務目的。 |
| 提交簽核 | 填寫完畢後送審。 |

##### 4.2.1.2 （a-2）核准

依公司規章中各項申請、請購、請修、請款、報銷核決權限表規定，填具請購申請單經所長核准後採買。

| 條件 | 規定 |
|---|---|
| 新台幣 30,000 元（含）以下 | 得以零用金方式支付。 |
| 超過新台幣 30,000 元 | 應由公司集中支付，或專簽以零用支付。 |
| 核准要求 | 所長核准後才可採買。 |

##### 4.2.1.3 （a-3）採買

任何採買均應取得載有本公司名稱、統一編號等合法之適當憑證（如發票、收據或其他）。若無法取得合法憑證或遺失憑證，得以支出證明單代替，並經單位主管核准；或至原商店要求影印發票存根，並加蓋店家發票章代替。

##### 4.2.1.4 （a-4）支出憑證審核

零用金保管人應負責審核憑證之合法性及正確性，審核項目如下：

| 審核項目 | 說明 |
|---|---|
| 1 | 金額是否合乎零用金支付限額之規定。 |
| 2 | 核章是否齊全。 |
| 3 | 支出憑證是否完備。 |
| 4 | 購買貨品應索取三聯式統一發票之扣抵聯及收執聯；如取得收銀機開立之統一發票收執聯，應加印影本當作扣抵聯。 |
| 5 | 統一發票須載明交易日期、品名、數量、金額、稅額及總計。 |
| 6 | 審核合格後，通知申請人領款，支付零用金後應隨即於相關憑證上蓋「付訖」章，以免重複付款。 |

###### 4.2.1.4.1 發票與收據審核補充

| 類型 | 審核重點 |
|---|---|
| 三聯式發票 | 注意公司名稱、統編、日期、統一發票專用章、大寫金額、付訖章。若資料錯誤不可塗改，需重新開立。 |
| 加油站發票 | 需辨識 22 類（載有稅額之其他憑證）與 25 類（三聯式收銀機發票）格式差異。 |
| 收銀機發票影本 | 可加印影本作為扣抵聯。 |
| 收據 | 如以現金支付，收據需貼千分之四印花；收款人資料需正確，若為電腦登打需親簽；支票到期日與支票號碼需載明。 |
| 免用統一發票收據 | 應蓋收據專用章及負責人章。 |
| 電子發票 | 自 104 年起 25 類統一發票定義變更，登打時需注意發票號碼與廠商統編。 |

##### 4.2.1.5 （a-5）報銷資料製作

固定零用金至少每半月應報銷一次，報銷時應檢附報銷清單連同原始憑證，經所長核印後送會計處核銷。另於報銷時，應附上當期零用金收支記錄影本 1 份，以供核對零用金動支狀況。

##### 4.2.1.6 （a-6）至（a-9）補充規定

| 編號 | 說明 |
|---|---|
| a-6 | 工務所應憑會計處支出傳票方可至工地銀行戶頭領款，完成領款後支出傳票送回會計處製作現金結存日報表。 |
| a-7 | 特定支付之大額款項或經主管單位核准指定款項，得專案申請臨時零用金支出；實際支付後應隨即專案檢據核銷及沖銷。若與原申請金額有差額，則多退少補。 |
| a-7-1 | 多退：臨時零用金多於實際報銷金額，多餘部分應填具存解款項清單繳回公司。 |
| a-7-2 | 少補：臨時零用金少於實際報銷金額，不足部分先行代墊後再專案報銷及請款，回補零用金。 |
| a-8 | 秉於日事日清原則，各單位領用零用金者，應於取得憑證當日送交零用金保管人辦理彙整。 |
| a-9 | 公差或公出報銷之辦理方式及核銷作業依公司營建管理資訊系統之「出差旅費報告表」系統辦理。 |

#### 4.2.2 （b）請款回補流程

| 步驟 | 說明 |
|---|---|
| b-1 工地請款 | 依支出傳票辦理回補請款，重點欄位包含請款金額、工令別、現金結存數、電匯工務所戶頭資料。 |
| b-2 出納撥款 | 財務或出納單位依流程撥款。 |
| b-3 工地銀行戶頭 | 回補款撥入工地銀行戶頭。 |
| b-4 工地保管人提款 | 工地保管人自銀行戶頭提款，回補零用金後再循環使用。 |

## 6. 其他事項

| 編號 | 說明 |
|---|---|
| 6.1 | 零用金保管人（或領用人）應設置「零用金收支登記簿」，將每日收支情形逐筆登記備查。 |
| 6.2 | 工務所收受買受憑證（如發票、收據等），若有相關核銷稅則規定及適法性疑義，洽公司會計處或財務處協助釋疑。 |
| 6.3 | 零用金一律存放於公司銀行專戶或各單位保管箱內；若因情況特殊無法依上述方式保管，需專簽核准。除不可抗力因素外，保管期間有任何短缺，由零用金保管者及單位主管負連帶賠償責任。 |
| 6.4 | 目前零用金請購仍維持於營管系統填單再匯入 EIP 系統電子表單簽核，核准後再進行採買，每次報銷時再進 PO-ERP 系統點收管控預算。 |
