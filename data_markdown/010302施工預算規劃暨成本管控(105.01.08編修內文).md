## 1. 目的

本公司所有承辦工程，在執行前需編製施工預算，以作為成本控制及收入與支出績效評估之基礎，並作為施工中相關項目成本與收入檢討之依據及變更管理之基準，藉以促進及提昇本公司競爭力。

## 2. 定義及範圍

### 2.1 定義

#### 2.1.1 直接成本

| 項目 | 說明 |
|---|---|
| 直接材料 | 指各種材料，執行時以購料案件辦理採購者。 |
| 設備類 | 指屬施工完成後為業主資產，如機電、消防工程。 |
| 直接人工 | 指自辦點工，包括領班（含）以下之現場實際施工人員。 |
| 機具車輛船舶 | 指各類機具車輛船舶費用，執行時可以租賃案件辦理採購，或辦理購置。 |
| 施工器材 | 指各類施工器材費用。公司內部撥用部分依施工器材管理辦法攤計費，或以租賃案件辦理採購，或辦理購置。 |
| 工務採購（工作服務） | 指執行時必須辦理之工程、勞務、作頭案件之採購費用。 |

#### 2.1.2 間接成本

| 項目 | 說明 |
|---|---|
| 工地直接費用 | 指直接使用於該工程，但須綜合分析者，及其他可直接歸屬於該工程之費用。 |
| 工安費用 | 指施工中為維護勞工安全衛生所需發生之費用。 |
| 環保費用 | 指因應環境保護，消除、防治施工噪音、水、空氣等污染所需費用。 |
| 睦鄰費用 | 指為順利完成工程而與周圍之人、事、物發生之相關費用。 |
| 工地間接費用 | 指工務所於施工期間估計發生而無法直接歸屬於某項工程之各項間接費用。 |
| 風險費 | 指工程施工期間風險費用。 |

**補充說明：工地直接費用範例**  
工地用人費用、工地事務費用、電腦設備採購、數位相機採購、工務所租地費用、辦公室及周邊設施、宿舍及周邊設施、圍籬、地磅、機具租賃等。

#### 2.1.3 施工預算表

| 項目 | 說明 |
|---|---|
| 施工預算彙總表 | 將工程合約總價，以及各類成本總計與分年度彙計分別填列。 |
| 施工計畫概要表 | 指工程主要項目施工計畫之概要。 |
| 五項直接成本之明細表與彙計表 | 直接成本明細表按合約項目分別計算；直接成本彙計表依資源項目彙總種類、數量與金額。 |
| 五項間接成本之明細表 | 按間接成本種類詳細分析計算各類資源之實際需要種類、數量及金額。 |
| 直接成本分析表 | 就各工程項目列出直接材料、直接人工、機具車輛船舶、施工器材、工務採購等成本金額。 |
| 單價分析表 | 按工作明細分析各工程項目之資源種類、使用量及價錢。 |
| 施工預計進度表 | 以線狀圖表示施工預定進度，並分月列明單月與累計的預定進度百分比及完成金額。 |

## 4. 流程及作業說明

## 4.1 施工預算編製流程（PDCA）

### 4.1.1 流程摘要

1. 工程得標。
2. 編製施工預算。
3. 召開預算審查會並核定。
4. 施工預算核定後由工務所執行。
5. 以預算資料持續進行管控與盈虧分析。
6. 發生差異時採取改善與修正行動。
7. 工程完工後辦理竣工結算與績效報告。

### 4.1.2 重要規則

| 項目 | 說明 |
|---|---|
| 主編單位 | 估算成控處負責主編；土木工程單價由採購發包處負責下標價，建築工程由估算成控處負責下標價。 |
| 工務所角色 | 工務所所長須與估算單位充分討論預算可行性，重點在工務所費用及施工／材料數量。 |
| 免編預算條件 | 總金額 500 萬以下，或交辦後二個月內完工之工程，得免編製施工預算。 |
| 核定方式 | 經工務所所長與估算成控處討論後，召集預算審查會，邀集相關單位訂定，呈總經理核定。 |
| 執行與監督 | 施工預算核定後由工務所切實執行及控制，由估算成控處負責控管及監督。 |
| 盈虧分析 | 以成本管制表等資料進行預算執行分析。 |
| 竣工結算 | 工程完工並經業主驗收確定後二個月內，應編報工程結算報告績效。 |

## 4.3 預算成本控制流程圖

### 4.3.1 管控主軸

| 主軸 | 說明 |
|---|---|
| 採購分包 | 以採購預算及成本管制表追蹤分包成本差異。 |
| 現金差異 | 透過會計月報、工程進度、現金回收期程預估與稽催機制追蹤。 |
| 成本差異 | 透過在建工程盈虧統計表、鋼筋物調統計表、財務管控分析表追蹤。 |
| 改善行動 | 針對差異原因提出改善及解決方案，視需要修正採購預算或施工預算。 |
| 專案列管 | 對持續異常或短期無法改善者進行專案列管與持續監控。 |

## 4.4 採購分包流程

### 4.4.1 主要規則

1. 工務所每三個月應統計填報「採購預算及成本管制表」及「大宗材料管制表」。
2. 應於次月 15 日前提送估算成控處研判評估各工令預算執行成效。
3. 分包成本負差異可由採購預算及成本管制總表中的標餘款為負值判斷。
4. 差異原因說明與可能盈虧預判，可於營建管理資訊系統的各工令財務管控分析表填報。
5. 必要時可修正採購預算，並持續監控。

### 4.4.2 採購預算及成本管制表相關表單

| 表單 | 用途 |
|---|---|
| 採購預算及成本管制總表 | 由明細表自動計算彙總結果。 |
| 採購預算及成本管制明細表 | 記錄合約編號、名稱、等級、時程、預算金額、決標金額、標餘款、廠商等。 |
| 業主變更設計採購合約管制表 | 追蹤採購合約受業主變更設計影響之預算、決標、追加減等。 |
| 業主契約變更管制表 | 追蹤契約變更號、理由內容、辦理情形、變更前後金額。 |
| 契約變更列入爭議採購合約管制表 | 追蹤已列入爭議之採購合約資訊。 |
| 保險理賠合約管制表 | 追蹤保險理賠造成之追加減與相關差異。 |

## 4.5 現金差異流程

### 4.5.1 主要規則

1. 工務所每月填報「在建工程盈虧統計表」、「鋼筋物調統計表」，回傳估算成控處綜合評效分析。
2. 工務所每月 25 日前應依當月實際施作完成金額提報工程進度表。
3. 會計處依工程進度表編製會計月報。
4. 會計處完成上月會計帳後，工務所應於下月 5 日上營建管理系統財務管控分析表填報資料。
5. 現金負差異與差異原因說明，可於各工令財務管控分析表填報。
6. 若現金短期無法回收，需進行紅燈警訊列管、稽催、改善方案、成效追蹤與持續監控。

## 4.6 成本差異流程

### 4.6.1 主要規則

1. 工務所每月填報在建工程盈虧統計表與鋼筋物調統計表。
2. 鋼筋物調統計表用於填報進度、未計價金額、尚未採購鋼筋重量，以及後續盈虧預估。
3. 成本負差異與差異原因說明可於各工令財務管控分析表填報。
4. 依差異金額、原因與盈虧預判決定是否重估工令成本。
5. 若需調整，則修正施工預算；若暫不調整，仍應專案列管與持續監控。
