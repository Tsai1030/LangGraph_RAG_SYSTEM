## 1. 目的

為有效管理電腦相關業務及維護資訊應用之安全，由本公司資訊處統籌購置、領用或移轉及網路連線規劃、軟體系統開發、軟硬體維修、教育訓練等業務之執行與控管。

## 2. 定義及範圍

### 2.1 定義

- **硬體**：各型電腦主機、網路設施、周邊設備之統稱。
- **軟體**：控制各項電腦硬體之操作程式、套裝程式、自行開發之應用系統程式及其他安全監控維護程式之統稱。
- **媒體**：指磁碟、磁片、磁帶、光碟、報表等儲存電腦資料用之媒介物。
- **檔案**：指儲存於各項電腦媒體中之所有程式及資料個別內容實體。

### 2.2 範圍

工務所購置之桌上型電腦、筆記型電腦、網路連結系統、印表機、掃描機、投影機、數據機、電腦處理主機及公司套裝軟體等均屬之。

## 4. 流程及作業說明

## 4.1 購置及維修申請（電子表單）流程及作業說明（含範例）

### 4.1.1 流程總覽

| 階段 | 說明 |
|---|---|
| 洽詢庫存 | 先向資訊處或其他單位確認是否有可調撥庫存。 |
| 購置申請（含預算） | 由工務所提出請購需求與預算。 |
| 初核 | 主管處審核。 |
| 會辦 | 資訊處及相關部門會辦。 |
| 核定 | 部門主管核定。 |
| 採購 | 資訊處辦理軟硬體採購。 |
| 安裝與測試 | 資訊處完成安裝及測試。 |
| 撥交及保管 | 驗收後撥交工務所保管。 |
| 維修申請 | 正常使用中如發生異常則提出維修。 |
| 維修更新 | 維修後更新狀態並恢復使用。 |

### 4.1.2 作業原則

#### (a) 庫存優先調撥

由使用單位先電洽資訊或其他單位有無庫存；若有庫存，優先逕予調撥。

#### (b) 預算編列與採購

資訊處應預先估算購置費用，由工務所據以編列預算，經主管處轉陳核定後，交由資訊處招商採購。

### 4.1.3 電子表單操作說明

#### (c) 進入系統

1. 進入中華工程企業入口網站：`https://beseip.bes.com.tw`
2. 輸入 Google 帳號及密碼。
3. 點選「營管系統」。
4. 於資訊查詢清單中點選「請購單系統」。
5. 進入請購單清單畫面後，點選「新增請購單」。

#### (c-3) 申請事項選項

依需求用途於下拉選單點選下列項目：

- 機械設備修繕及電腦修繕
- 3C 產品（電腦、數位相機等）及週邊設備
- 電腦軟體服務費
- 購置得以費用列支之雜項購置（列管資產）

#### (c-3-1) 必填欄位

| 欄位 | 說明 |
|---|---|
| 申請事項 | 必填 |
| 用途說明 | 必填 |
| 物品名稱 | 必填 |
| 請購數量 | 必填 |
| 單價 | 必填 |

補充說明：

- 可使用「新增請購項目」新增細項。
- 可先點選「儲存草稿」保留下次續編。
- 請購單位及請購人員預設為填單人。
- 因業務需求可代填請購單，但簽核層級仍依請購單位與請購人員身分由系統判斷。

#### (c-3-2) 未送簽狀態

- 狀態為「未送簽」時，可點選「請購單號」開啟請購單並編輯。
- 若該筆請購項目數為 0，可直接刪除。

#### (c-4) 提交簽核

請購單編輯完成後，點選「提交簽核」，將表單送入簽核流程。

#### (c-5) 送出確認

送出簽核時，系統會跳出確認對話框；點選「確定」後開啟新視窗顯示簽核畫面。

#### (c-6) 簽核處理

- 在流程單下方點選「同意」可送至下一位簽核人。
- 也可選擇加簽人員後點選「同意」辦理會簽。

#### (c-6-1) 流程中狀態

請購單清單畫面中，該筆資料狀態顯示為「流程中」，並會顯示流程單號。

#### (c-6-2) 流程中查詢限制

請購單於簽核流程中，點選請購單號僅供查詢，不能修改內容。

#### (c-6-3) 已核決

當核決人簽核並點選「同意」後，該請購單在清單中會顯示「已核決」。

#### (c-7) 已駁回

如請購單在流程中被駁回，請購單清單會顯示「已駁回」。

#### (c-7-1) 駁回後重送

駁回的請購單可點選請購單號重新進入表單修正內容，再次提交簽核，並產生新的流程單號。

### 4.1.4 核定後作業

#### (d) 安裝及測試

資訊處至工務所協助安裝及測試，驗收合格後撥交工務所保管使用。

#### (e) 資產列管

資訊處於電腦撥交時，應依公司規定納入列管資產，並辦理資產移轉領用手續。

## 4.2 電腦相關業務服務申請單：使用權限申請流程及作業說明（含範例）

### 4.2.1 流程總覽

| 階段 | 說明 |
|---|---|
| 使用權限申請 | 工務所提出權限申請。 |
| 初核 | 申請單位主管初核。 |
| 核定 | 資訊處或權責單位核定。 |
| 修改完成通知 | 完成後通知申請人。 |
| 連線操作 | 依申請內容完成系統權限或連線操作。 |
| 處理結果通知 | 將處理結果回覆申請單位。 |
| 結案 | 流程結束。 |

### 4.2.2 電子表單操作說明

1. 進入企業入口網站。
2. 點選「電子簽核」。
3. 進入電子表單系統。
4. 點選「（新）資訊系統權限申請單」。
5. 於「資訊系統權限申請單」中填寫相關欄位。
6. 點選「送出表單」將表單送入簽核流程。

### 4.2.3 權限申請表單重點欄位

| 欄位 | 說明 |
|---|---|
| 連絡電話 | 需填寫 |
| 營管系統功能面 | 勾選申請之權限 |
| 工令及業務內容說明 | 說明申請背景 |
| 中工圖書館角色 | 工務所申請需由工務處審核 |
| 其他系統說明 | 非屬既有項目時於此補充 |
| POERP 系統權限 | 需參閱 POERP 系統權限設定表 |
| 附件 | 視申請內容附上必要附件 |
| 單位主管 | 填寫單位審核主管 |

### 4.2.4 簽核流程

| 順序 | 角色 | 說明 |
|---|---|---|
| 1 | 申請單位主管 | 初核 |
| 2 | 資訊單位 | 簽辦 |
| 3 | 權責主管 | 核定 |
| 4 | 系統/資訊單位 | 完成設定並結案 |

## 6. 其他事項

### 6.1 軟體合法性

為遵守著作權法，嚴禁同仁使用非法軟體。電腦操作管理人應切結保證並同意公司所定之使用電腦軟體政策。

### 6.2 保密義務

資訊應用系統設計者、電腦網路管理者或負責處理機密資料者，應依規定簽署保密協定；離職時應繳回所有通行證，並取消電腦設備之使用識別碼及通行碼。
