## 1. 目的

工務所依承辦工程契約規定及用人預算，得經主管處同意後進用人員，或由其他工務所、公司內相關單位調派人員，以執行交派工作或預定任務。

## 2. 定義及範圍

### 2.1 適用範圍

- 適用期間：承辦工程動員開工至竣工驗收施工期間之進用及調派。

## 3. 公司規程

1. 本公司「營建工程管理作業流程準則」工程管理03：工務所人員進用與調派作業流程。
2. 本公司「同仁任用與升遷辦法」。
3. 本公司「工作同仁薪給管理要點」。

## 4. 流程及作業說明

### 4.1 流程總覽

| 流程階段 | 說明 |
|---|---|
| 提出人力需求 | 工務所依工程執行需求提出人力需求。 |
| 初核 | 主管處與人力處共同研商，優先評估內部調派可行性。 |
| 調派或任用 | 若可內部調派，走調派程序；若無適當人員，改由人力處召聘與任用。 |
| 核定 | 由相關主管單位核定後通知工務所辦理進用。 |
| 工務所進用 | 進用後辦理勞健保與到職管理。 |
| 工地人員管理 | 包含考勤、遲到早退、加班、生活照顧、獎懲與所長評核。 |

### 4.2 提出人力需求與初核

#### 4.2.1 (a) 提出人力需求

工務所應就工程執行需求與主管處商討後，提出實際人力需求。

#### 4.2.2 (a) 初核原則

主管處與人力處研商初核後，優先由其他工務所或公司內相關單位調派適合人員；若無適當人員，則移會人力處依規定召聘人員。

#### 4.2.3 (a-1) 調派人員

| 項目 | 內容 |
|---|---|
| 適用情境 | 優先採內部調派方式補足人力 |
| 辦理方式 | 至電子公文填寫「工作調動單」 |
| 主管調動補充規定 | 如為主管調動，需先上簽呈，奉核後再進行人員調派流程 |
| 關鍵表單 | 工作調動單 |

#### 4.2.4 (a-2) 申請進用公司聘用人員

| 表單 | 用途 |
|---|---|
| 附表一：人力需求申請表（公司人員適用） | 提出公司聘用人員需求 |
| 附表二：人力需求條件表 | 補充需求條件與工作內容 |

填寫重點：

- 工作內容應詳實填寫清楚。
- 需求原因應詳實填寫清楚。

#### 4.2.5 (a-3) 申請進用工地聘用人員

| 表單 | 用途 |
|---|---|
| 附表三：人力需求申請表（定期契約人員適用） | 申請進用工地聘用或定期契約人員 |

### 4.3 申請任用

#### 4.3.1 (b) 申請任用

當人力需求確認需採任用方式處理時，進入任用申請程序。

#### 4.3.2 (b-1) 工地進用公司聘用人員

| 項目 | 內容 |
|---|---|
| 表單 | 附表四：進用人員簽辦表（公司人員適用） |
| 送件流程 | 送人力處核轉主管處核定 |
| 審核重點 | 應確認進用人員學經歷並填上正確資料 |

#### 4.3.3 (b-2) 進用工地聘用人員

| 項目 | 內容 |
|---|---|
| 表單 | 附表五：工務所進用定期契約人員核備表 |
| 用途 | 辦理工地聘用或定期契約人員之核備 |

### 4.4 核定與進用

#### 4.4.1 (c) 工務所進用

奉核定後，由人力處通知工務所進用。工務所應依規定辦理勞健保。

### 4.5 工地人員管理

#### 4.5.1 (d-1) 正常工時與考勤

| 項目 | 規定 |
|---|---|
| 正常工時 | 每日 8 小時 |
| 上午上班時間 | 08:30 至 12:30 |
| 下午上班時間 | 13:30 至 17:30 |
| 工地彈性 | 因工作需要得調整上下班時間 |
| 考勤方式 | 可設置簽到簿、退簿、打卡鐘，或混合方式辦理 |
| 遲到標準 | 逾上班 15 分鐘到公且未及簽到者視為遲到 |
| 遲到處理 | 除應簽到外，並應以請假方式處理不足時數 |
| 管理要求 | 管理考勤人員應即查考登錄出勤遲到狀況 |

#### 4.5.2 (d-2) 遲到、早退與曠職

| 狀況 | 處理原則 |
|---|---|
| 逾上班半小時到公 | 除因公或另經准假者外，均以曠職論處 |
| 因故早退 | 除因公或另經准假者外，均以曠職論處 |
| 每月遲到三次且未補辦請假 | 以曠職論 |

#### 4.5.3 (d-3) 加班管理

| 項目 | 規定 |
|---|---|
| 加班適用條件 | 主管指派或臨時急要交辦工作，且必須利用正常工時外完成 |
| 每日上限 | 不得超過 4 小時 |
| 每月總上限 | 不得超過 46 小時 |
| 補休期限 | 應於次年 6 月底前休畢 |
| 管理原則 | 事前核備，事後簽核 |

#### 4.5.4 (d-4) 生活照顧

工地同仁之辦公、住宿、膳食等，於工務所奉准成立後，依工程合約規定設置組織，並採覓地自建（組合屋）或租賃辦公場所方式辦理；對遠到同仁得視實際狀況安排住宿、膳食等生活照顧。

#### 4.5.5 (d-5) 獎勵與榮譽

工地同仁進駐工區後，應戮力完成工務所指派任務並爭取榮譽，並得依下列規定申請額外實質獎勵：

1. 公司「品質管理獎懲辦法」
2. 「工務所額外盈餘獎勵辦法」
3. 「本公司勞工安全衛生獎懲辦法」

#### 4.5.6 (d-6) 所長每季評核制度

為有效管理工地各項事務正常運作，並防範所長發生異常狀態影響工地執行效率，特建立所長每季評核制度，以強化所長工地管理。

##### 4.5.6.1 評等標準

| 評等 | 分數區間 | 處理原則 |
|---|---|---|
| A 等 | 80 分以上 | 工地管理尚稱良善，持續觀察 |
| B 等 | 80 分～65 分 | 工地需加強管理，增加查證頻率並給予輔導及協助；連續 2 次 B 等須提報總經理並檢討是否更換所長；若不宜更換，則指派專案經理駐地督導 |
| C 等 | 65 分以下 | 列為重大異常工地處理 |
