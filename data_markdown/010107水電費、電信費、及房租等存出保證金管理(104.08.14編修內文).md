## 文件摘要

本文件說明工務所在 **水電費、電信費、房租等存出保證金** 的支付、核銷、保管與收回作業。重點包括：

- 保證金屬於資產，不是費用。
- 支付保證金時，需依流程申請工程週轉金或以小額零用金支付。
- 核銷時，保證金收據正本送會計處，影本由工務所專人保管。
- 工務所遷移、合併、裁撤或完工時，應辦理解約並收回保證金，再依規定繳回公司並完成帳務處理。

## 1. 目的

為有效管理公司資產，特訂此存出保證金作業規範。

## 2. 定義、範圍及保管

### 2.1 定義

存出保證金是指存出供作保證用途的現金或其他資產，性質屬於資產，不是費用，也就是押金或保證金。

### 2.2 範圍

適用範圍包括但不限於以下項目：

- 給付給電力公司的用電保證金
- 給付給自來水公司的水費押金
- 租用土地或房屋所付的保證金
- 租用郵局信箱的押金
- 其他以公司名義、以現金對外給付、具保證性質且可提回的現金

### 2.3 保管

| 項目 | 規定 |
|---|---|
| 保證金單據正本 | 送回會計處核銷，並附於傳票之後 |
| 保證金單據影本 | 由工務所指派專人保管 |
| 所長或保管人異動 | 影本單據需列入移交清單 |
| 工務所合併 | 影本單據需列入移交清單 |
| 工務所裁撤或工程完工 | 依 3.2 收回保證金流程辦理 |

## 3. 流程及作業說明

## 3.1 支付保證金流程

### 3.1.1 流程總覽

| 流程階段 | 說明 |
|---|---|
| 工務所發生租押金 | 因施工需要發生水電、電信、房租等押金或保證金 |
| 申請工程週轉金 / 工務所零用金申請表 | 依金額與情境申請撥款 |
| 會計處帳務處理 | 會計處先行作帳 |
| 財務處開立支票 / 匯款工務所帳戶 | 財務處撥付資金 |
| 工務所支付保證金 | 對外支付押金或保證金 |
| 工務所取具收據正本單獨核銷 | 不入 POERP 系統點收，影本由工務所專人保管 |
| 會計處帳務處理 | 完成核銷與沖帳 |

### 3.1.2 作業說明

#### (a) 一般支付保證金流程

工務所因施工需要發生租押金時，奉核後填寫 **工程週轉金請款單** 或 **工務所零用金申請表** 向會計處請款。  
會計處作帳後，財務處開立支票或匯款給工務所。  
工務所支付保證金後，需取具 **收據正本**，並單獨製作一份報銷送會計處核銷。

注意事項：

- 保證金報銷 **不入 POERP 系統點收核銷**
- 收據 **正本** 送會計處核銷
- 收據 **影本** 由工務所專人保管
- 保管資料需列入移交

#### (a-1) 保證金簽文

辦理保證金申請時，需有核准簽文，作為支付與請款依據。

#### (a-2) 工程週轉金請款單

申請週轉金時，需填寫：

- 請款金額
- 工令別

#### (a-3) 會計帳務處理（通知出納匯款）

會計處完成帳務處理後，通知出納辦理匯款。

#### (a-4) 財務處開立支票或匯款工務所帳戶

財務處依核准文件，將款項以支票或匯款方式撥入工務所帳戶。

#### (a-5) 工務所取具收據報銷

工務所支付保證金後，需進行報銷，重點如下：

| 項目 | 說明 |
|---|---|
| 報銷方式 | 以收據正本單獨核銷 |
| POERP 點收 | 不入 POERP 系統點收 |
| 影本保存 | 影本由工務所專人保管 |
| 系統作業 | 於營建管理系統登打報銷作業並列印報銷彙總表 |
| 關聯欄位 | 需記錄報銷彙總表號與預先請款單號 |

#### (a-6) 會計處帳務處理（沖預先請款）

會計處依據報銷資料與預先請款單進行沖帳。

### 3.1.3 小額保證金以零用金支付

#### (b) 以工地零用金支付小額保證金

若工務所因施工需要發生租押金，且屬小額情形，可先以 **工地零用金** 支付。  
支付後，再取具收據正本單獨製作一份報銷，向會計處辦理核銷，以補回零用金。

注意事項：

- 仍屬 **不入 POERP 系統點收核銷**
- 影本由工務所專人保管
- 本流程常見於 **自來水保證金** 等小額項目

#### (b-1) 工務所取具收據報銷

辦理小額保證金報銷時，需注意：

- 報銷單據需標示保證金項目
- 影本保存地點需明確
- 可於單據上標註保管人資訊

#### (b-2) 會計處帳務處理（支出傳票）

會計處開立支出傳票後，工務所憑支出傳票至工地銀行戶頭領款。  
傳票上需填寫：

- 領款日期
- 領款日簽章

## 3.2 收回保證金流程

### 3.2.1 流程總覽

| 流程階段 | 說明 |
|---|---|
| 工務所遷移 / 合併 / 裁撤 / 完工 | 已無續租或續用必要 |
| 工務所至會計處取回附於傳票後正本收據 | 取回原始憑據以辦理解約與退款 |
| 向機關或房東申請收回保證金 | 以支票或電匯回公司帳戶 |
| 填寫存解款項清單並送交財務處 | 作為繳回與帳務依據 |
| 會計處帳務處理 | 沖減保證金科目 |

### 3.2.2 作業說明

當工務所因遷移、合併、裁撤或工程完工而已無續租必要時，須指派專人：

1. 向機關或房東辦理解約。
2. 收回保證金。
3. 若收回形式為支票或現金，須儘速寄回或電匯回公司財務處。
4. 填寫 **存解款項清單** 通知財務處。
5. 財務處確認收訖後，由會計處依清單辦理帳務。

### 3.2.3 填寫存解款項清單重點

| 欄位 | 說明 |
|---|---|
| 繳回方式 | 註明支票、匯款或其他方式 |
| 保證金項目 | 註明繳回何種保證金 |
| 金額 | 註明保證金金額 |
| 匯費 | 註明是否含匯費 |
| 繳款人 | 需簽章 |
| 電匯總公司銀行帳號 | 應填寫正確帳號 |

### 3.2.4 會計處帳務處理

會計處收到存解款項清單後，進行 **沖減銀行保證金科目** 的帳務處理。
