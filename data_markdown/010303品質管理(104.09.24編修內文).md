# 010303 品質管理

## 文件中繼資料

- 文件名稱：品質管理
- 作業流程節次：010303
- 版本日期：104.09.24 編修內文
- 來源類型：工務所管理作業手冊
- RAG 文件策略：保留原始章節、流程、定義、權責、材料設備品質管制流程、檢(試)驗結果管制與文件清單，並補充圖像索引與表格結構化欄位。

## 1. 目的

為引導工務所全員參與品質管理活動，落實公司品質政策目標，建立品質管理程序，以達成公司品質要求、業主契約規定或顧客滿意。

## 2. 定義及範圍

### 2.1 範圍

1. 擬訂品質計畫書並推動工務所品質制度運轉。
2. 依業主契約規範、自建案合約規範、品質計畫書、分項工程施工計畫書、施工圖、自主查驗表、樣品與實品戶製作等，執行現場施工品質要求與檢查作業。
3. 產品或材料進場時進行品質管控，必要時得會同業主或供應者共同辦理檢(試)驗。

### 2.2 定義

#### 2.2.1 品質管理
為公司品管組織或專案業主品管組織最上層所為之品管行為，包括品質政策與目標制定、推廣宣導、教育訓練、品質管理文件製作與修訂、抽查品質、評獎處分。

#### 2.2.2 品質保證
為工地品質督導或業主委託顧問監造所為之品質管制作業督導、檢驗審核或稽查。

#### 2.2.3 品質管控
為工地現場工程師或專案第一線執行單位（如承包商）為確保完成工作產品達到規範要求之日常檢驗作為。

#### 2.2.4 檢(試)驗
針對施工中或已完成之分項作業施工品質（含使用材料或產品），驗證其是否符合規範或圖說要求。

#### 2.2.5 不符合
不能滿足契約規範或圖說之要求。

#### 2.2.6 不合格品
經檢(試)驗或測試過程，發現品質不符合要求之文件、材料、儀器、設備或施工產品。

#### 2.2.7 檢驗停留點（限止點 hold point）
工作進行過程中經相關單位指定之停留點；該點工作未經檢(試)驗或同意，不得進行後續工作。工作到達停留點前，應依規定通知相關單位檢(試)驗日期、時間與地點。

#### 2.2.8 自主檢查之檢驗點
針對特定工作項目施工成果加以檢查的時機點。

## 3. 流程及作業說明

### 3.1 權責層級

#### 3.1.1 總公司品質管理處（品質管理）

1. 協助、監督及審查工務所制訂品質計畫書及作業程序書，以符合契約規範及總公司品質管理手冊規定。
2. 擬定總公司年度內部品質稽核計畫，對工務所各單位作定期內部稽核。
3. 督導辦理年度品質管理教育訓練。
4. 維護 ISO-9001 品質管理系統並輔導工務所接受 ISO 外部稽核。
5. 維護與更新公司品質管理手冊內容。

#### 3.1.2 工務所所長／副所長（品質管理）

1. 制訂工務所作業程序書，經總公司品質管理處審查核定後公佈並要求同仁簽名確認。
2. 協助、監督與審查品管站制訂品質計畫書，以及施工站制訂各分項工程施工計畫書、施工圖、自主查驗表、樣品與實品戶製作等。
3. 督導工務所同仁依品質計畫書及各分項工程施工計畫書等規定確實執行。
4. 配合業主、顧客（自建案）或品質管理處接受定期或不定期工程查核。
5. 配合總公司年度內部品質稽核計畫，接受定期或不定期內部稽核。

#### 3.1.3 品管站站長（品質保證）

1. 制訂品質計畫書、材料設備檢試驗管制流程、施工品質管制及品質統計分析。
2. 制訂不合格品管制及矯正預防措施處理程序。
3. 擬定工務所年度內部品質稽核計畫。
4. 擬定品質管理教育訓練。
5. 監督產品材料設備檢(試)驗與施工品質查驗。
6. 監督、協助品質工程師制定品質計畫書。
7. 監督現場工程師依規定落實自主品質管理。
8. 接受業主、顧客或品質管理處工程查核。
9. 接受總公司內部稽核。

#### 3.1.4 品管工程師（品質保證）

1. 依工程契約、設計圖說、施工技術規範及品質計畫製作綱要制定品質計畫書並推動執行。
2. 執行內部品質稽核，檢核自主檢查表內容及結果是否確實記錄。
3. 執行品質技術文件記錄管制並妥善保存。
4. 落實品管統計分析、矯正與預防措施之提出及追蹤。
5. 會同業主、供應者或顧客辦理產品材料設備檢(試)驗及施工品質查驗。
6. 追蹤及管制不合格品，避免誤用，並依規定辦理缺失矯正及預防措施。
7. 接受工程查核與內部稽核。
8. 稽核現場工程師或協力廠商是否依規定執行並記錄。
9. 辦理其他提升工程品質之相關事宜。

#### 3.1.5 現場工程師（品質管控）

1. 依契約、品質計畫書及各分項工程施工計畫書規定執行施工品質作業。
2. 依分項工程施工計畫書落實施工自主檢查並填寫自主檢查表。
3. 辦理現場施工品質管理業務，並執行瑕疵或缺失矯正改善措施。
4. 會同業主、顧客或供應者辦理產品材料設備檢(試)驗及施工品質查驗。
5. 管理施工品質技術文件紀錄。

### 3.2 流程

#### 3.2.1 材料設備供應管制作業流程

- 流程主軸：尋覓材料設備供應者 → 資格遴選及詢價評估 → 依「國內採購作業辦法」辦理採購／發包 → 材料設備供應廠商資格審查 → 補充相關證明文件 → 合約履行及管理 → 材料設備訂貨及下單
- 標記說明：
  - `★`：檢驗停留點（限止點）
  - `◆`：自主檢查檢驗點

##### 材料設備供應管制重點

1. 投標前應清楚核對專案工程契約、圖說、規範之品質要求，以及材料設備規格、數量與時程。
2. 尋覓供應者時，應考量合法登記、服務績效、品質管制、交貨時間與配合度。
3. 依「國內採購作業辦法」辦理採購與發包。
4. 供應商應依合約要求提送公司執照、營業登記證、材料規範等品質證明文件。
5. 材料設備供應廠商資格審查後，如不足，應補充相關證明文件。

##### 資格審查所需資料

| 項目 | 說明 |
|---|---|
| 材料型式、類別、等級 | 提供精確鑑別說明文件 |
| 材料檢(試)驗資料 | 各種檢(試)驗及適用規格 |
| 品質系統標準 | 本工程採用之品質系統標準 |
| 其他特殊規定 | 依契約或專案需求提供 |

#### 3.2.2 材料設備進場前管制作業

工程材料設備審查合格後，在進料使用前，應依備料時間、進場時間及檢(試)驗時機適時管制。

##### (1) 備料時間

- 國內採購與國外採購的備料時間不同，國外採購需預留更長備料時間。
- 採購原則以整批採購、分批進料方式辦理，以降低成本並減少待料。

##### (2) 進場時間

- 各項材料進場前，應依規定提出檢(試)驗申請程序。
- 應依工程進度規劃進場，並安排妥當且安全之堆置場所。

##### (3) 管制時機

- 依契約規範或圖說，訂定出廠前訪廠、廠驗及送檢(試)驗項目。
- 確認是否需提送廠測計畫書或其他相關計畫書。
- 參照施工預定進度推算產品供應時程，正式通知相關單位派員會同辦理。

##### (4) 作業程序

1. 工務所應明確掌握需辦理檢(試)驗之材料設備，詳細研讀契約規範。
2. 若國內、外皆無試驗機構，應正式函文相關單位釐清，避免影響驗收。
3. 應提送符合 CNS 規定之 TAF 實驗室登錄證明文件送備查。
4. 核備後依工程進度辦理產品檢(試)驗。

#### 3.2.3 材料設備檢(試)驗作業流程

- 流程主軸：材料設備訂貨及下單 → 產品進場前準備 → 廠驗／訪廠 → 材料設備進場 → 材料設備進場檢(試)驗 → 複驗結果評估 → 施工安裝或存放／拒收退貨

##### 流程判斷邏輯

| 階段 | 符合 | 不符合 |
|---|---|---|
| 廠驗或訪廠 | 進入後續進場流程 | 瑕疵改善及複驗 |
| 進場檢(試)驗 | 允收，進入施工安裝或存放 | 依不合格品程序辦理 |
| 複驗 | 合格則允收 | 不可複驗則退貨運離工地 |

##### 作業要點

1. 產品進場前應依合約與工程進度需求完成訂貨及下單。
2. 進場時應備妥符合 CNS 或契約規範之 TAF 實驗室與相關證明文件。
3. 產品檢試驗不符合時：
   - 不可複驗：拒收、退貨、運離工地。
   - 可複驗：辦理瑕疵改善後再複驗。
4. 檢驗合格並允收後，依施工進度及作業環境辦理安裝或存放。

#### 3.2.4 材料設備進場後管制作業

1. 材料設備進場後，原則上應妥善安全堆置於工作面附近，以利施工。
2. 工務所應預先規劃材料設備待驗區、加工區、廢料區等區域。
3. 如有平行標案施工界面動線障礙，應於 CIP（Coordinated Installation Program）協調會議中處理。
4. 待驗區應依材料設備特性、種類作防雨、防撞、防塵等保護措施。
5. 應檢附相關產品證明文件並依規定提出材料進場檢驗申請。
6. 合格品應標示為合格品，並依預定時程進場施工安裝。
7. 不合格品應依缺失改善追蹤機制及「不合格品管制程序」辦理。
8. 如發現主要缺失或同類型瑕疵重複發生，應積極提出矯正與預防措施。

#### 3.2.5 材料設備檢(試)驗結果管制作業

##### (1) 符合

- 檢(試)驗合格後，始得施工安裝。
- 除記錄於自主檢查表外，亦應考量工程估驗、完工驗收、請照或竣工報告之留存需求。

##### (2) 不符合

- 檢(試)驗結果不符合設計標準或規範要求時，列為不良品。
- 應於材料堆置區明顯標示為不良品，不得使用。
- 由品管人員登錄並依「不合格品管制程序」辦理：
  - 識別
  - 缺失判定
  - 改善
  - 不合格材料管制處理
  - 缺失改正與矯正措施
  - 追蹤留存記錄與回饋
- 在未經檢(試)驗合格及允許前，不得使用。
