## 1. 目的

本流程為使工務所於施工期間，得以最短時程完成對業主估驗及計價，進而跟催款項核撥入帳，俾利公司資金有效調度及現金收支均衡調節。

## 2. 定義及範圍

### 2.1 定義

| 編號 | 名稱 | 說明 |
|---|---|---|
| 2.1.1 | 檢驗停留點 | 分項施工程序中，顯有影響後續施工品質或事後查證困難之虞，經業主（監造）指定停留檢驗者；作業達停留點前，應以書面方式通知業主（監造）檢驗日期、時間、地點，非經同意不得進行後續作業。 |
| 2.1.2 | 自主檢查表 | 分項施工之自主檢查及停留點檢驗等，應按契約規定之施工及材料檢驗程序與檢驗項目及允收標準訂定；檢驗結果應量化記錄，並由業主（監造）會驗人員簽署。 |
| 2.1.3 | 計價資料 | 業主規定製作之計價文件及其相關佐證資料，如數量計算、施工照片、檢驗表單、試驗報告等。 |

### 2.1.4 審查權責分工

| 權責單位 | 檢核事項 |
|---|---|
| 施工站 | 確認施工檢驗停留點符合業主（監造）要求並留存簽核之自主檢查表；提供計價資料（數量計算、施工照片及檢驗表單等）；日、月報數量統計及填報；檢核設計數量與施工數量差異。 |
| 品管站 | 提供材料檢試驗報告資料。 |
| 內業站 | 製作日、月報表；儘速辦理契約數量變更追加減；檢查契約條件符合度並彙總計價資料；追蹤業主計價審查與核撥入帳進度並排除審查疑義。 |
| 副所長 | 覆核、督導、整合相關計價申辦資料，必要時主動介入與業主溝通。 |
| 所長 | 核定、督導、整合相關計價申辦資料，必要時主動介入與業主溝通。 |

## 3. 流程及作業說明

### 3.1 業主估驗計價流程總覽

工務所作為與總公司作為之主要流程如下：

1. 分項施工前確認業主（監造）之檢驗停留點，並列入自主檢查表。
2. 施作完成工項、數量、表單，現場會同業主（監造）檢驗簽認。
3. 每日跟催工程施工照片及各類施工紀錄。
4. 清圖檢核設計數量與契約數量（現場施作數量）差異，儘速辦理變更追加減。
5. 施工數量填入日、月報。
6. 製作計價資料並提計價申請。
7. 計價數量與比例先對業主溝通說明。
8. 完成工審後納入計價。
9. 業主預審相關計價資料。
10. 預審合格後，函送紙本並掣開發票。
11. 計價資料送審（業主流程）。
12. 跟催核撥入帳。

### 3.2 分項施工前與自主檢查表

#### 3.2.1 提送自主檢查表

依品質計劃書或相關施工規範規定，製作自主檢查表等相關施工查核表格，提送業主審查。

#### 3.2.2 製作自主檢查表（檢驗停留點）

自主檢查表應明確列出：
- 檢驗停留點
- 公司工程師查核欄位
- 施工廠商查核欄位
- 會驗簽署欄位

#### 3.2.3 業主會同查核作業

工項施作完成後，業主工程師應會同查核，並於自主檢查表或相關表單上簽認。

#### 3.2.4 完工審查資料彙整

將自主檢查表、業主查核表及相關計價需求資料彙整後，提送完工審查。

### 3.3 日報、月報與計價資料製作

#### 3.3.1 月報填寫

業主填入監工月報，統計完成數量。

#### 3.3.2 製作計價資料

彙整當月完成之施作數量後，填入計價估驗系統，製作計價詳細表。

### 3.4 完成工審計價與預審

#### 3.4.1 完成工審計價

計價數量與計價比例，應先對業主溝通說明；完成計價資料後提送完工計價審驗，並協助業主審核相關計價資料。

#### 3.4.2 業主預審

業主、監造及工務所先行預審核相關計價資料。

#### 3.4.3 掣開計價發票

確認預審無誤後，計價資料即可用印，並掣開計價發票。

### 3.5 計價送審與入帳

#### 3.5.1 計價資料送審

業主內部相關單位（例如會計部門）於撥款前審查，並辦理相關計價撥款核章流程。

#### 3.5.2 計價金額入帳

工務所應持續跟催業主計價流程，至核撥入帳。

## 5. 其他事項

### 5.1 縮短業主估驗計價技巧說明

| 編號 | 技巧 |
|---|---|
| 5.1.1 | 計價前與業主（監造）召開計價協議會議，訂定標準計價流程及應備文件；建築工程需訂定各工項分層分棟計價比例。 |
| 5.1.2 | 建立計價相關人員聯絡單，專責估驗工程師應設有職務代理人。 |
| 5.1.3 | 計價數量先與業主（監造）溝通，說明計算方式及公式，以取得共識。 |
| 5.1.4 | 佐以圖說說明本次施工計價範圍與已施工計價範圍，以作未計價範圍內控。 |
| 5.1.5 | 以當期累積數量－上期累積數量＝本期新增數量。 |
| 5.1.6 | 計價數量無條件捨去取至整數，小數位累積至下期，以避免誤差累計。 |
| 5.1.7 | 若實作數量與規劃數量差異超乎預期，應即與監造討論並辦理變更或修正。 |
| 5.1.8 | 與業主（監造）承辦人員誠信互動，掌握時程，避免因小額細項估驗延誤總額估驗。 |

### 5.2 現金與計價管理

#### 5.2.1 管理重點

對業主計價之現金管理非常重要，會影響公司週轉、利息負擔及淨利，應列入專案管理績效考核。

#### 5.2.2 計價缺口（差異）公式

```text
計價缺口（差異）＝當月廠商計價發票累計金額 ÷ (1－當月實際毛利率) － 當月業主計價發票累計金額
```

#### 5.2.3 造成計價缺口之主要因素

| 代號 | 因素 |
|---|---|
| a | 業主與廠商計價時間差與條件差異 |
| b | 業主辦理變更尚未完成，公司需先墊付 |
| c | 業主階段性完成後計價，小包按月計價 |
| d | 業主合約數量少於實作數量 |
| e | 業主因預算不足或會計年度因素延遲計價 |
| f | 工務所內部作業緩慢、文件不完整、品質不良遭擋款 |
| g | 公司內部調撥沖銷或物價調整扣款等因素 |

#### 5.2.4 議價與計價標準管理

專案開始前議價階段，應儘量就各種工種、材料、機具、設備、工作與服務之計價標準與定義討論清楚，避免爭議。

#### 5.2.5 業主延遲計價付款時之建議

1. 平時做好工作文件整理與存檔。
2. 對業主或監造之口頭指示，應即製作備忘錄。
3. 計價文件務求正確清楚，並附當月施作照片。
4. 可先送草稿與業主監造單位溝通，達成預先協議。
5. 少數爭議項目可暫時擱置或延至下期，以免影響整筆計價。
6. 簽認核准之計價文件為重要爭議依據。
7. 對延遲付款情形，應積極跟催。
8. 投標前應將業主付款風險納入考量。
9. 對惡性延遲付款，停工或法律措施須以合約站得住腳。

#### 5.2.6 收款動作建議

| 編號 | 作法 |
|---|---|
| a | 每周召開財務會議，透明化各專案收款情形並預測未來三個月應收帳款。 |
| b | 收到現金或支票才可認列為收入。 |
| c | 將遲收計價款所造成之利息損失納入專案績效考核。 |
| d | 動員前應先就合約金額、單價、付款條件與業主達成原則性協議。 |

#### 5.2.7 驗收與催尾款策略與流程

| 步驟 | 作法 |
|---|---|
| a | 請業主監造詳列缺失清單。 |
| b | 修改完成後，可分項逐次請業主驗收。 |
| c | 行文給業主請領尾款。 |
| d | 如未獲合理回應，可視情況向法院申請支付命令。 |
| e | 驗收簽署文件對尾款結清與履約保證之取回極為重要，務必儘早取得。 |
