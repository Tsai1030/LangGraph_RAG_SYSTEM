## 1. 目的

為有效運用及管理本公司資產，除法令及本公司會計制度另有規定外，所有固定及列管資產之購置、營繕、改良及擴充均納入作業規範。

## 2. 定義及範圍

### 2.1 定義

#### 2.1.1 固定資產

固定資產包括土地、土地改良物、房屋建築、機械設備、交通運輸設備、什項設備，且需同時符合下列條件：

- 各資產取得價值逾新台幣捌萬元
- 使用年限逾兩年

#### 2.1.2 列管資產

以費用或材料支出，而不以固定資產列帳之機械設備（含施工器具、試檢驗儀器、電腦周邊設備、生活用電器設備等）、交通運輸設備及什項設備，經經管單位、使用單位或財務單位認定有列管必要者。

#### 2.1.3 經管單位

| 單位 | 資產經管項目 |
|---|---|
| 採發處 | 機械設備（電腦設備除外）、交通運輸設備（總公司公務車輛及電信設備除外） |
| 資訊處 | 電腦設備 |
| 企資處 | 土地、土地改良物、房屋建築、電信設備、什項設備、總公司公務車輛 |

### 2.2 範圍

資產取得、領用、異動、保管、折舊、投保、盤點、投資抵減、報廢及處分均適用之。

## 4. 流程及作業說明

## 4.1 購置及使用流程

### 4.1.1 流程摘要

購置及使用流程可概括為：

1. 購置前洽經管單位確認是否有庫存
2. 無庫存時提出購置申請
3. 主管處核定
4. 辦理採購
5. 驗收
6. 辦理固定或列管資產登錄
7. 領用或移轉
8. 正常使用
9. 定期盤點
10. 如有資產盈虧則調查原因並調整列帳

### 4.1.2 購置前洽經管單位

工務所於購置資產前，應先行電洽資產經管單位，確認該資產無庫存後，始得申請購置。

目前資產請購方式為：

- 先於營建管理系統填單
- 再匯入 EIP 系統電子表單簽核
- 核准後進行採購

### 4.1.3 無庫存時之請購流程

無庫存時，進入營建管理系統「請購單系統」填寫表單，主要步驟如下：

1. 進入系統填單
2. 點選新增請購單
3. 選擇請購單位
4. 選擇請購人員及所屬工令
5. 選擇申請事項
6. 填寫用途說明
7. 填寫請購項目
8. 提交簽核

### 4.1.4 核定與否

#### 核定原則

購置申請需由主管處核定後，依本公司採購規定辦理。

#### 核定後處理

- 請購單經主管處核定後，辦理採購及安裝。

#### 核否後處理

- 請購單經主管處核否後，需重新填寫請購單。

### 4.1.5 採購

由資產管理單位決定：

- 由資產管理單位自行辦理採購，或
- 交由工務所自行辦理採購

每次報銷時，再進 PO-ERP 系統點收並管控預算。

### 4.1.6 驗收

資產採購交貨時，除授權工務所自行驗收者外，應由經管單位會同工務所驗收，並製作資產標誌撥交工務所使用。

點收完成後，工務所應依本公司資產管理規定，送交驗收單至資產管理單位，辦理納入固定或列管資產。

#### 資產驗收單重點

(d-2) 提送資產驗收單時，應檢附：

- 統一發票
- 器材照片

並回送總公司經管單位編列資產。

### 4.1.7 固定或列管資產登錄

資產領用、使用、保管、異動及報廢，均應依公司規定辦理。

由資產管理單位辦理：

- 納入固定或列管資產
- 辦理資產移轉領用手續

### 4.1.8 領用或移轉

#### 移轉流程

由移出單位填寫資產移轉領用單，交移入單位用印後，送交資產管理單位辦理登錄。

#### 新品領用及標誌

資產管理單位由資產管理系統編列資產編號，並印製資產移轉領用單一式三聯：

- 一聯送交領用單位
- 領用單位依規定保管與使用

### 4.1.9 資產驗收單欄位說明

| 欄位 | 填寫說明 |
|---|---|
| 合約編號、廠商編號及合約交貨日期 | 有辦理採購合約時需填寫 |
| 實際交貨日期 | 交貨安裝完成日 |
| 驗收編單號 | 由工務所自行編列流水號 |
| 驗收日期 | 以工務所實際完成驗收日期為準 |
| 收貨單位 | 申請使用之工務所 |
| 發票號碼 | 實際發票所列號碼 |
| 資產編號 | 由總公司管理單位填寫 |
| 其餘欄位 | 依實際品項填列 |

## 4.2 資產標誌樣式與規則

### 4.2.1 採發處經管資產使用之資產標誌

資產標誌牌欄位如下：

| 欄位 | 內容 |
|---|---|
| 資產編碼 | 資產唯一識別碼 |
| 品名規格 | 至少應明列廠牌、型號等 |
| 購入原值 | 以新台幣元為單位 |
| 購入年月 | 統一以西元年月日表示 |
| 使用年限 | 依資產規定填列 |

#### 尺寸規格

| 尺寸代號 | 規格 |
|---|---|
| A | W80 × H40 × T3 mm |
| B | W140 × H60 × T3 mm |
| C | W200 × H120 × T3 mm |

#### 製作與標示原則

- 上述 A、B、C 三種尺寸，視資產規格及特性由使用單位自行決定製作。
- 大型機具、器材應另以噴漆標示資產編號。

### 4.2.2 企資處經管資產（雜項設備）使用之資產標誌

欄位如下：

| 欄位 | 內容 |
|---|---|
| 資產編號 | 資產唯一識別碼 |
| 品名規格 | 品名與規格 |
| 購置日期價格 | 購置日期與價格 |

### 4.2.3 資訊處經管資產（電腦設備）使用之資產標誌

欄位如下：

| 欄位 | 內容 |
|---|---|
| 資產名稱 | 設備名稱 |
| 入帳年月 | 入帳日期 |
| 資產編號 | 資產編號 |
| 原值 | 原始價值 |
| 規格 | 主要規格 |
| 保固年限 | 保固期限 |

#### 尺寸

- 尺寸：9 × 1.8 cm

### 4.2.4 資產標誌共通規則

- 日期統一以西元年月日表示，例如：2002.08.07
- 價格統一以新台幣元為單位，例如：NT$100,000
- 品名規格至少應明列廠牌、型號等
- 資產標誌固定方式以貼紙固定於資產上

## 4.3 定期盤點

### 4.3.1 盤點時點

資產使用期間，工務所應於每年五月三十日前，自資產管理系統資料庫查詢前三個月資產數量及現況，回報經管單位。

### 4.3.2 盤點責任

經管單位應會同工務所辦理資產盤點至少每年一次。

若盤點發現資產盈或虧，應立即：

1. 查明原因
2. 調整列帳

### 4.3.3 系統操作重點

進入營建管理系統「固定資產查詢及使用狀況更新列印」，並完成下列作業：

1. 點選固定資產查詢及使用狀況更新
2. 選擇使用部門及資產類別
3. 點選列印固定資產
4. 確實填寫存放地點、現況、使用人
5. 單位主管簽章後送資產管理單位

## 4.4 報廢流程

### 4.4.1 流程摘要

報廢流程可概括為：

1. 資產逾使用期限
2. 申請報廢
3. 檢視是否堪用
4. 若堪用則入庫統一調度
5. 若不堪用則判斷是否具經濟價值
6. 不具經濟價值者採最小成本處理
7. 具經濟價值者辦理變賣

### 4.4.2 資產逾使用期限

資產逾使用期限者，即可列入報廢評估對象。

### 4.4.3 申請報廢

清點逾使用年限資產，提出報廢申請。

### 4.4.4 堪用與否

檢視待報廢資產是否仍堪使用。

### 4.4.5 庫存

若資產尚堪使用，由工務所簽報經管單位依專業判定後，使用單位應填具移轉單，先行入庫，再由經管單位憑以統一調度。

### 4.4.6 具經濟價值

不堪使用者，需再檢視報廢資產是否具經濟價值。

### 4.4.7 最小成本處理

#### 不具經濟價值者

由資產管理人進公司營管系統填寫固定／列管資產損毀報廢單，經主管用印後，檢附資產照片，送交總公司經管單位辦理。

#### 固定／列管資產損毀報廢單重點欄位

| 類別 | 欄位 |
|---|---|
| 基本資料 | 資產編號、名稱、購置原價 |
| 使用資訊 | 使用起迄日期、已提折舊金額、殘值金額、報損金額 |
| 原因與處分 | 損毀原因、擬處分辦法 |
| 簽章 | 由資產管理人、站長、主管簽章 |

#### 後續作業

1. 收到經管單位簽報奉准報廢文件後，至有關主管機關辦理報廢手續。
2. 應洽合法登記之回收商以最小成本處理。

### 4.4.8 變賣

#### 適用情形

具經濟價值者，於完成報廢手續後，變賣出售予資源回收商。

#### 款項處理

所得款項應繳交總公司財務處開立發票交予回收商。

#### 發票重點欄位

| 欄位 | 內容 |
|---|---|
| 買受人名稱 | 回收商名稱 |
| 統一編號 | 回收商統編 |
| 地址 | 回收商地址 |
| 品名 | 變賣品名 |
| 金額 | 銷售金額 |

#### 結案

完成報廢及變賣出售等手續後，檢送報廢資料及出售統一發票影本，交總公司經管單位辦理除帳，並結案建檔存查。

## 6. 其他事項

### 6.1 移轉運什費

移出單位之資產因移轉所需之運什費，由接管單位負擔。

### 6.2 工務所合併

被合併單位應陳交「資產清單暨使用狀況表」予合併單位，合併單位將影本送經管單位更新資料。

### 6.3 工務所撤銷或不使用資產

工務所承辦人員應確實清點資產，辦理退庫或移轉予其他單位使用。

### 6.4 保管責任

各使用單位應指派專管人員負責資產操作或使用。原則上：

- 領用人為保管人
- 各單位主管為監管人

### 6.5 日常核對義務

資產使用期間，各使用單位及其保管人應經常於公司資產管理系統核對保管資產情形是否有誤；如發現差異，應隨即向經管單位查明及更正。
