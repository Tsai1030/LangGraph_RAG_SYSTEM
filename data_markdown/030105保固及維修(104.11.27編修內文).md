## 內容摘要

本文件說明工程竣工驗收後之保固與維修作業，包含保固責任起算、保固業務交接、分期分項保固期限確認、業主通知保固維修、現場會勘、責任廠商或公司維修分流、施工完成後驗收會勘、保固期滿解除保固，以及最後一期保固金退還或銀行保證責任解除等流程。並附有捷運站與醫院案例、常見缺失事項及處理方式，適合作為保固維修管理與售後服務之 RAG 檢索資料。

## 1. 目的

因應本公司承攬工程於竣工驗收後之保固規定，訂定內部保固及維修作業標準，以確保售後服務品質及樹立公司商譽形象。

## 2. 定義及範圍

竣工驗收合格後，依業主契約保固規定辦理下列作業：

- 通知
- 會勘
- 維修
- 結案

## 3. 流程及作業說明

## 3.1 保固及維修流程摘要

1. 工程驗收合格
2. 點交使用單位
3. 工務所撤銷，保固業務交付指定單位
4. 確認分期、分項保固期限
5. 業主保固維修通知
6. 損壞會勘
7. 確認是否屬保固項目
8. 區分責任廠商維修項目或公司維修項目
9. 估價、預算、核定、選商發包或通知責任廠商進場
10. 現場施工
11. 施工完成、驗收會勘、維修紀錄
12. 發函結案
13. 保固屆滿發函業主解除保固
14. 缺失改善完成後辦理驗收會勘
15. 請業主退最後一期保固金或解除銀行保證責任
16. 工程結案

## 4. 工程驗收合格與保固起算

### 4.1 保固起算基準

- 工程驗收合格取得「工程結算驗收證明書」後，
- 自驗收合格日期開始起算保固。

### 4.2 點交使用單位

- 依照合約規定辦理點交。
- 點交作業可依本手冊第 030101 節「完成品保護與點交」辦理。
- 應包含點交清冊製作與移交資料整理。

## 5. 保固業務交付指定單位

### 5.1 原工務所移交資料

原執行工務所應移轉下列資料供後續保固作業使用：

- 協力廠商名冊
  - 公司名稱
  - 分包項目
  - 公司住址
  - 聯絡人
  - 電話
- 施工照片
- 竣工圖說
- 竣工結算明細
- 驗收證明書
- 驗收及點交紀錄
- 工程變更設計加減帳資料
- 公共設施操作維護手冊
- 教育訓練資料
- 其他必要文件

### 5.2 確認保固範圍與期限

- 應確認保固範圍、內容及期限。
- 依業主契約規定確認各分期、分項保固期限。
- 屆滿次日應發函業主解除該期保固責任，並退還該期保固金或解除銀行保證責任。
- 最終保固期限屆滿次日，應發函業主解除全工程保固責任，並退最後一期保固金或解除銀行保證責任。

## 6. 業主通知保固維修與責任判斷

### 6.1 通知方式

- 業主多先以電話或電子郵件通知，
- 隨後再正式發函通知。
- 保固期限屆滿前，業主通常會辦理保固會勘，以確認解除保固責任前之維修事項。

### 6.2 現場會勘與記錄

- 本公司人員應現場勘查損壞情形並作成記錄。
- 若非屬本公司保固責任範圍：
  - 應具體詳實、委婉說明
  - 再發函結案
- 若屬本公司保固責任範圍：
  - 應於期限內完成維修
  - 再發函結案

### 6.3 維修責任確認

分為兩類：

1. **責任廠商維修項目**
2. **公司維修項目**

## 7. 責任廠商維修流程

### 7.1 通知責任廠商

- 先電話通知，再正式發函要求責任廠商於期限內進場維修。
- 辦理與業主會勘時，應約定責任廠商一同到場，以縮短維修時程。

### 7.2 廠商是否進場

#### 7.2.1 廠商確定進場維修
- 約定進場日期。
- 進場前須電話聯絡業主或使用單位確認施工日期與時間。
- 並確認進場前所有必要手續。

#### 7.2.2 廠商不進場維修
- 公司得逕行發包另請承商維修。
- 所需工料費（含管理費）自保固金扣款。
- 並發函通知責任廠商。

### 7.3 維修完成後

- 施工完成後應辦理：
  - 驗收會勘
  - 維修紀錄
  - 函覆結案

## 8. 公司維修流程

### 8.1 啟動條件

下列情形由公司自行辦理維修：

- 屬公司維修項目
- 已通知責任廠商但廠商不進場維修

### 8.2 公司辦理步驟

1. 確認維修項目
2. 估價及預算
3. 簽辦、核定
4. 選商發包（依採購發包作業流程）

### 8.3 進場施工注意事項

- 進場前須先電話聯絡業主（使用單位）約定施工日期與時間。
- 須確認進場前必辦手續。
- 私人建案進入住家維修時，應注意：
  - 服裝儀容
  - 修繕前各項保護
    - 電梯
    - 地面
    - 傢俱
- 避免造成二次傷害或污染。

### 8.4 維修完成後

- 辦理施工完成確認
- 驗收會勘
- 記錄維修內容
- 函覆結案

> 文件建議所有修繕應納入客服系統，以完整記錄客訴與修繕來龍去脈。

## 9. 保固屆滿與解除保固

### 9.1 發函解除保固

- 保固屆滿次日應發函業主解除保固。
- 有些業主要求必須由公司正式發函。

### 9.2 解除保固前之缺失改善

- 業主通常於保固期滿時通知保固檢驗會勘。
- 對缺失改善完成後，業主再通知保固檢驗複驗。
- 複驗合格後，應請業主：
  - 退最後一期保固金
  - 或解除銀行保證責任
- 完成後工程結案。

## 10. 案例：高雄車站（捷運 R11 臨時站）保固維修

### 10.1 案例背景

- 結構工程保固 5 年
- 本標工程代辦機關為交通部鐵路改建工程局
- 驗收後移交高雄捷運公司使用

### 10.2 業主與使用單位通知維修

- 業主及使用單位通知缺失維修項目。
- 工務所聯絡鐵工局、高雄捷運公司及保固廠商約定會勘日期。
- 現場會勘確認施作方法與施工日期時間。

### 10.3 施工限制

高雄捷運公司維修要求包括：

1. 保固維修多安排於夜間施工
2. 若屬軌道管制區，須在確認電車供電系統斷電後方可進入
3. 所有進場人員必須接受 6 小時教育訓練並領有識別證
4. 須事先申請進場門禁管制單
5. 動火作業須事先申請
6. 可能影響火警感知器之作業須先做好包覆並施工後復原
7. 影響消防系統時須申請消防系統暫時隔離
8. 進入軌道區前必須取得行控中心授權碼
9. 確認已斷電後，依標準完成接地始可施工
10. 施工完成後須通報人員退出管制區並歸還授權碼

### 10.4 保固期滿會勘缺失項目示例

文件舉例之缺失包括：

- 上行三軌編號 U061 處地面積水
- 上行北端三軌編號 U032 處柔性接頭滲水
- 上行三軌編號 U017 處側牆滲水
- 下行三軌編號 D04 處截水盤引水需處理

### 10.5 觀察型缺失之處理

- 若保固期滿後仍有滲水或需觀察一段時間之缺失，
- 可與業主協商以「切結書」方式處理，
- 以利及早解除保固並領回保固金或解除銀行保證責任。

## 11. 案例：國立成功大學醫學院附設醫院保固維修

### 11.1 保固年限

- 結構體保固 5 年
- 水電及裝修保固 3 年

### 11.2 施工限制

- 醫院已在營運中
- 保固維修時間通常規定於：
  - 星期六下午
  - 星期日休診時

### 11.3 維修完成後

- 缺失改善後須照相
- 並經使用部門簽認
- 再發函通知業主保固缺失已改善完成

## 12. 常見缺失事項與處理方式

### 12.1 建築、結構工程常見缺失

| 缺失項目 | 處理方式 |
|---|---|
| 公共空間滲漏水 | 須優先進行緊急事件處理，再進行後續維修 |
| 地面層外牆滲漏水 | 常以窗台或裂縫填補，外牆不易由外施作時改由內側止水 |
| 地下層外牆滲漏水 | 常用鑽孔打發泡劑止水，施工前須確認牆內無管線 |
| 頂版滲漏水 | 若位於電氣機房須優先處理，並可設接水盤 |
| 地下層地板滲水 | 止水無效時可改採導水方式 |
| 衛浴給排水設備故障 | 請水電承商檢修 |
| 門窗鎖故障 | 請門窗廠商檢修或由保固人員自行調整 |
| 供電系統故障 | 請專業廠商或技術人員檢修 |
| 空調系統故障 | 請專業廠商或技術人員檢修 |
| 地坪損壞或裂縫 | 若影響通行安全須優先處理 |
| 牆壁油漆裂縫 | 若業主無急迫性，可於保固期滿前統一處理 |
| 景觀植栽樹木枯死 | 請景觀植栽廠商進行保固補植 |

### 12.2 道路、橋梁工程常見缺失

| 缺失項目 | 處理方式 |
|---|---|
| 道路路基凹陷、AC 龜裂 | 切割損壞處，挖除柏油，路基回填滾壓夯實後鋪 AC |
| 道路 AC 路面滑動 | 同上，切割挖除並重鋪 AC 路面 |
| 道路 AC 路面出油 | 同上，切割挖除並重鋪 AC 路面 |
| 標線磨損、標誌鬆脫 | 通知專業廠商排定日期進場改善 |
| 橋面伸縫處損壞 | 通知專業廠商排定日期進場改善 |
| 混凝土護欄龜裂 | 通知專業廠商排定日期進場改善 |
| 排水系統故障 | 通知專業廠商排定日期進場改善 |
| 機電系統故障（號誌、路燈） | 通知專業廠商排定日期進場改善 |
| 景觀植栽樹木枯死 | 請景觀植栽廠商進行保固補植 |
